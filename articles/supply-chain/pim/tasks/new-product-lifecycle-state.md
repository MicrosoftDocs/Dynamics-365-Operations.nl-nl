# <a name="create-a-new-product-lifecycle-state"></a>Een nieuwe status voor de productlevenscyclus maken

[!include [banner](../../includes/banner.md)]

Deze procedure laat zien hoe u een nieuwe status voor een productlevenscyclus maakt. Een levenscyclusstatus documenteert de levenscyclusstatus van een vrijgegeven product of productvariant. Statussen voor de productlevenscyclus worden gedefinieerd door de gebruiker, doorgaans een productmanager of een manager van producthoofdgegevens. Specifieke bedrijfsprocessen, zoals de hoofdplanning, kunnen worden be√Ønvloed door een specifieke levenscyclusstatus.


## <a name="create-a-product-lifecycle-state"></a>Een status voor de productlevenscyclus maken
1. Ga naar Productgegevensbeheer > Instellen > Levenscyclusstatus van product.
2. Klik op Nieuw.
3. Typ een waarde in het veld Status.
4. Typ een waarde in het veld Omschrijving.
