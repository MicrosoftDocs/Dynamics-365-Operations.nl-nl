# <a name="define-configuration-groups"></a>Configuratiegroepen definiëren

[!include [task guide banner](../../includes/task-guide-banner.md)]

Deze procedure laat zien hoe u productconfiguratiegroepen definieert door twee groepen te maken in het demobedrijf USMF. Dit is de vierde van acht procedures waarin wordt uitgelegd hoe u combinaties maakt voor een op dimensies gebaseerde configuratie.

1. Ga naar Productgegevensbeheer > Instellingen > Stuklijsten en formules > Configuratiegroepen.
2. Klik op Nieuw.
3. Typ een waarde in het veld Configuratiegroep.
    * Typ een naam voor de configuratiegroep die voor de stuklijstregels wordt gebruikt.  
4. Typ een waarde in het veld Naam.
5. Klik op Nieuw.
6. Typ een waarde in het veld Configuratiegroep.
    * Typ een naam voor de configuratiegroep die voor de stuklijstregels wordt gebruikt.  
7. Typ een waarde in het veld Naam.
8. Sluit de pagina.
