# <a name="functional-location-lifecycle-states"></a><span data-ttu-id="b3d3f-103">Levenscyclusstatussen van functionele locatie</span><span class="sxs-lookup"><span data-stu-id="b3d3f-103">Functional location lifecycle states</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="b3d3f-104">In dit onderwerp wordt beschreven hoe u levenscyclusstatussen van functionele locaties en levenscyclusmodellen instelt in Activabeheer.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-104">This topic describes how to set up functional location lifecycle states and lifecycle models in Asset Management.</span></span> <span data-ttu-id="b3d3f-105">Levenscyclusstatussen van functionele locaties definiëren de statussen die een functionele locatie kan doorlopen, bijvoorbeeld gemaakt, actief en beëindigd.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-105">Functional location lifecycle states define the states that a functional location can go through, for example, created, active, and ended.</span></span> <span data-ttu-id="b3d3f-106">U kunt alle functionele locaties weergeven, ongeacht hun levenscyclusstatus, op de lijstpagina **Alle functionele locaties**.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-106">You are able to view all functional locations, regardless of their lifecycle state, in the **All functional locations** list page.</span></span> <span data-ttu-id="b3d3f-107">U kunt de status van een functionele locatie wijzigen door deze te selecteren op de lijstpagina **Alle functionele locaties** en **Status van functionele locatie bijwerken** te selecteren.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-107">You can change the state of a functional location by selecting it in the **All functional locations** list page and selecting **Update functional location state**.</span></span>

## <a name="set-up-functional-location-lifecycle-states"></a><span data-ttu-id="b3d3f-108">Levenscyclusstatussen van functionele locatie instellen</span><span class="sxs-lookup"><span data-stu-id="b3d3f-108">Set up functional location lifecycle states</span></span>

1. <span data-ttu-id="b3d3f-109">Selecteer **Activabeheer** > **Instellingen** > **Functionele locaties** > **Levenscyclusstatussen**.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-109">Select **Asset management** > **Setup** > **Functional locations** > **Lifecycle states**.</span></span>
2. <span data-ttu-id="b3d3f-110">Selecteer **Nieuw** om een nieuwe status voor een functionele locatie te maken.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-110">Select **New** to create a new functional location state.</span></span>
3. <span data-ttu-id="b3d3f-111">Voeg de status-id in het veld **Levenscyclusstatus** in en geef een naam op voor de status van de functionele locatie in het veld **Naam**.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-111">Insert the state ID in the **Lifecycle state** field and a name for the functional location state in the **Name** field.</span></span> <span data-ttu-id="b3d3f-112">In het veld **Levenscyclusmodellen** kunt u het aantal levenscyclusmodellen voor functionele locatie zien dat gebruikmaakt van de status van de functionele locatie.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-112">In the **Lifecycle models** field, you can see the number of functional location lifecycle models that uses the functional location state.</span></span>
4. <span data-ttu-id="b3d3f-113">Selecteer op het sneltabblad **Algemeen** de optie 'Ja' op de wisselknop **Actief** als de functionele locatie bij deze status actief moet zijn.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-113">On the **General** FastTab, select "Yes" on the **Active** toggle button if the functional location should be active at this state.</span></span>
5. <span data-ttu-id="b3d3f-114">Selecteer 'Ja' op de wisselknop **Activa maken** als het mogelijk moet zijn om automatisch een activum met dezelfde naam te maken als de functionele locatie en dit bij deze status te installeren op de functionele locatie.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-114">Select "Yes" on the **Create assets** toggle button if it should be possible to automatically create an asset with the same name as the functional location and install it on the functional location at this state.</span></span>  
>[!NOTE]
><span data-ttu-id="b3d3f-115">Deze wisselknop heeft betrekking op het veld **Activatype** op het sneltabblad **Algemeen** in het formulier **Typen functionele locatie** (**Activabeheer** > **Instellingen** > **Functionele locaties** > **Typen functionele locatie**).</span><span class="sxs-lookup"><span data-stu-id="b3d3f-115">This toggle button relates to the **Asset type** field on the **General** FastTab in the **Functional location types** form (**Asset management** > **Setup** > **Functional locations** > **Functional location types**).</span></span>
6. <span data-ttu-id="b3d3f-116">Selecteer 'Ja' op de wisselknop **Locatie hernoemen** als het mogelijk moet zijn om de naam van de functionele locatie bij deze status te wijzigen.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-116">Select "Yes" on the **Rename location** toggle button if it should be possible to change the name of the functional location at this state.</span></span>
7. <span data-ttu-id="b3d3f-117">Selecteer 'Ja' op de wisselknop **Nieuwe sublocaties** als het mogelijk moet zijn om nieuwe sublocaties toe te voegen aan de functionele locatie bij deze status.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-117">Select "Yes" on the **New sub locations** toggle button if it should be possible to add new sub locations to the functional location at this state.</span></span>
8. <span data-ttu-id="b3d3f-118">Selecteer 'Ja' op de wisselknop **Activa installeren** als het mogelijk moet zijn om activa te installeren op de functionele locatie bij deze status.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-118">Select "Yes" on the **Install assets** toggle button if it should be possible to install assets on the functional location at this state.</span></span>
9. <span data-ttu-id="b3d3f-119">Selecteer 'Ja' op de wisselknop **Functionele locatie verwijderen** als het mogelijk moet zijn om de functionele locatie te verwijderen bij deze status.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-119">Select "Yes" on the **Delete functional location** toggle button if it should be possible to delete the functional location at this state.</span></span>
10. <span data-ttu-id="b3d3f-120">Selecteer een activastatus in het veld **Levenscyclusstatus** als u wilt dat de levenscyclusstatus van het activum automatisch wordt bijgewerkt bij deze status voor alle activa die op de functionele locatie zijn geïnstalleerd.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-120">Select an asset state in the **Lifecycle state** field if you want the asset lifecycle state for all assets installed on the functional location to be automatically updated at this state.</span></span> <span data-ttu-id="b3d3f-121">Voorbeeld: als u een functionele locatie sluit en de levenscyclusstatus van de functionele locatie instelt op 'Beëindigd', wilt u mogelijk automatisch de levenscyclusstatus van de activa die op die functionele locatie zijn geïnstalleerd wijzigen in 'Niet in gebruik'.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-121">Example: If you close down a functional location, and set the functional location lifecycle state to "Ended", you may want to automatically change the lifecycle state of the assets installed on that functional location to "Not in use".</span></span>


>[!NOTE]
><span data-ttu-id="b3d3f-122">Levenscyclusstatussen van functionele locaties, levenscyclusmodellen voor functionele locaties en typen functionele locaties zijn gerelateerd en worden op dezelfde manier gebruikt als levenscyclusstatussen van werkorders, levenscyclusmodellen voor werkorders en typen werkorders.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-122">Functional location lifecycle states, lifecycle models, and types are related and used in the same way as work order lifecycle states, work order lifecycle models, and work order types.</span></span> 

## <a name="set-up-functional-location-lifecycle-models"></a><span data-ttu-id="b3d3f-123">Levenscyclusmodellen voor functionele locaties instellen</span><span class="sxs-lookup"><span data-stu-id="b3d3f-123">Set up functional location lifecycle models</span></span>

<span data-ttu-id="b3d3f-124">Wanneer u de levenscyclusstatussen hebt gemaakt die zijn vereist voor uw functionele locaties, kunnen deze worden onderverdeeld in groepen.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-124">When you have created the lifecycle states required for your functional locations, they can be divided into groups.</span></span> <span data-ttu-id="b3d3f-125">Dit wordt gedaan om de levenscyclusmodelstroom te maken die kan worden gebruikt voor verschillende typen functionele locaties.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-125">This is done to create the lifecycle model flow that may be used for different types of functional locations.</span></span> <span data-ttu-id="b3d3f-126">Er moet minimaal één standaard levenscyclusmodel voor functionele locaties worden gemaakt.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-126">As a minimum, one standard functional location lifecycle model should be created.</span></span>

1. <span data-ttu-id="b3d3f-127">Selecteer **Activabeheer** > **Instellingen** > **Functionele locaties** > **Levenscyclusmodellen**.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-127">Select **Asset management** > **Setup** > **Functional locations** > **Lifecycle models**.</span></span>
2. <span data-ttu-id="b3d3f-128">Selecteer **Nieuw** om een nieuw levenscyclusmodel te maken.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-128">Select **New** to create a new lifecycle model.</span></span>
3. <span data-ttu-id="b3d3f-129">Voeg de levenscyclusmodel-id in het veld **Levenscyclusmodel** in en geef een naam op voor het levenscyclusmodel in het veld **Naam**.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-129">Insert the lifecycle model ID in the **Lifecycle model** field and a name for the lifecycle model in the **Name** field.</span></span> <span data-ttu-id="b3d3f-130">In de velden **Typen functionele locatie** en **Levenscyclusstatussen** kunt u het aantal typen functionele locatie zien dat gebruikmaakt van het levenscyclusmodel en het aantal statussen dat is geselecteerd in het levenscyclusmodel.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-130">In the **Functional location types** and **Lifecycle states** fields, you can see the number of functional location types that uses the lifecycle model and the number of states selected in the lifecycle model.</span></span>
4. <span data-ttu-id="b3d3f-131">Selecteer op het sneltabblad **Levenscyclusstatussen** de statussen die in het model moeten worden opgenomen.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-131">On the **Lifecycle states** FastTab, select the states that should be included in the model.</span></span> <span data-ttu-id="b3d3f-132">Dit wordt gedaan door op een status te klikken in de sectie **Resterende levenscyclusstatussen** en op de knop ![pijl naar voren](media/02-setup-for-functional-locations.png) te klikken.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-132">This is done by clicking on a state in the **Lifecycle states remaining** section and clicking the ![forward arrow](media/02-setup-for-functional-locations.png) button.</span></span>
5. <span data-ttu-id="b3d3f-133">Als u alle beschikbare statussen voor een model wilt selecteren, klikt u op de knop ![Alle beschikbare fasen selecteren](media/03-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="b3d3f-133">If you want to select all the available states for a model, click the ![select all available stages](media/03-setup-for-functional-locations.png) button.</span></span> <span data-ttu-id="b3d3f-134">Alle statussen worden overgebracht naar de sectie **Geselecteerde levenscyclusstatussen**.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-134">All states are transferred to the **Lifecycle states selected** section.</span></span>
6. <span data-ttu-id="b3d3f-135">Als u een geselecteerde status uit het model wilt verwijderen, selecteert u de status in de sectie **Geselecteerde levenscyclusstatussen** en selecteert u vervolgens de knop ![pijl terug](media/04-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="b3d3f-135">If you want to remove a selected state from the model, select the state in the **Lifecycle states selected** section and then select the ![back arrow](media/04-setup-for-functional-locations.png) button.</span></span>
7. <span data-ttu-id="b3d3f-136">Selecteer **Updates van levenscyclusstatussen** om te definiëren welke levenscyclusstatussen een geselecteerde status kunnen volgen.</span><span class="sxs-lookup"><span data-stu-id="b3d3f-136">Select **Lifecycle state updates** to define which lifecycle states can follow a selected state.</span></span>
