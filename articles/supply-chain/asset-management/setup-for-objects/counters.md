<span data-ttu-id="8496a-104">In dit onderwerp wordt uitgelegd hoe u metingtypen voor activa maakt in Activabeheer.</span><span class="sxs-lookup"><span data-stu-id="8496a-104">The topic explains how to create asset measure types in Asset Management.</span></span> <span data-ttu-id="8496a-105">Metingtypen voor activa worden gebruikt om metingregistraties te maken voor activa, bijvoorbeeld met betrekking tot het aantal productieuren of de hoeveelheid die op het activum wordt geproduceerd.</span><span class="sxs-lookup"><span data-stu-id="8496a-105">Asset measure types are used to make measurement registrations on assets, for example, regarding number of production hours, or quantity produced on the asset.</span></span> <span data-ttu-id="8496a-106">De activatypen zijn gerelateerd aan de metingtypen voor activa.</span><span class="sxs-lookup"><span data-stu-id="8496a-106">Asset types are related to the asset measure types.</span></span> <span data-ttu-id="8496a-107">Dit betekent dat een activummeting alleen voor een activum kan worden gebruikt als de activummeting is ingesteld voor het activumtype dat met het activum wordt gebruikt.</span><span class="sxs-lookup"><span data-stu-id="8496a-107">This means that an asset measure can only be used on an asset if the asset measure is set up on the asset type used on the asset.</span></span>

<span data-ttu-id="8496a-112">Een registratie van activummeting kan handmatig of automatisch worden bijgewerkt op basis van productieuren of de geproduceerde hoeveelheid.</span><span class="sxs-lookup"><span data-stu-id="8496a-112">An asset measurement registration can be updated manually or automatically based on production hours or quantity produced.</span></span> <span data-ttu-id="8496a-113">Een activummeting kan worden ingesteld voor het gebruiken van een van deze drie bijwerkmethoden (geselecteerd in het veld **Bijwerken** in **Tellers**):</span><span class="sxs-lookup"><span data-stu-id="8496a-113">An asset measure can be set up to use one of three update methods (selected in the **Update** field in **Counters**):</span></span>

>[!NOTE]
><span data-ttu-id="8496a-117">Als geproduceerde hoeveelheid wordt gebruikt, worden *alle* geregistreerde artikelen opgenomen in de metingregistratie, zowel de goede artikelen als artikelen met fouten.</span><span class="sxs-lookup"><span data-stu-id="8496a-117">If quantity produced is used, *all* registered items are included in the measurement registration, good quantity as well as error quantity.</span></span> <span data-ttu-id="8496a-118">Het is altijd mogelijk om een handmatige registratie van de activummetingen te doen, als dat nodig is.</span><span class="sxs-lookup"><span data-stu-id="8496a-118">It is always possible to make manual asset measurement registrations, if required.</span></span>

## <a name="create-counter-types-for-asset-counter-registrations"></a><span data-ttu-id="8496a-119">Tellertypen maken voor registraties van activatellers</span><span class="sxs-lookup"><span data-stu-id="8496a-119">Create counter types for asset counter registrations</span></span>

2. <span data-ttu-id="8496a-121">Selecteer **Nieuw** om een nieuw metingtype voor activa te maken.</span><span class="sxs-lookup"><span data-stu-id="8496a-121">Select **New** to create a new asset measure type.</span></span>
3. <span data-ttu-id="8496a-122">Geef een id op in het veld **Teller** en een tellernaam in het veld **Naam**.</span><span class="sxs-lookup"><span data-stu-id="8496a-122">Insert an ID in the **Counter** field, and a counter name in the **Name** field.</span></span>
4. <span data-ttu-id="8496a-123">Selecteer op het sneltabblad **Algemeen** een maateenheid in het veld **Eenheid**.</span><span class="sxs-lookup"><span data-stu-id="8496a-123">On the **General** FastTab, select a measurement unit in the **Unit** field.</span></span>
5. <span data-ttu-id="8496a-124">Selecteer in het veld **Bijwerken** de bijwerkmethode die voor de activummeting moet worden gebruikt.</span><span class="sxs-lookup"><span data-stu-id="8496a-124">In the **Update** field, select the update method to be used for the asset measure.</span></span>
6. <span data-ttu-id="8496a-125">Selecteer "Ja" op de wisselknop **Tellerwaarden overnemen** als onderliggende activa in een activastructuur automatisch de geregistreerde activummeting moeten overnemen van het bovenliggende activum.</span><span class="sxs-lookup"><span data-stu-id="8496a-125">Select "Yes" on the **Inherit counter values** toggle button if child assets in an asset structure should automatically inherit asset measure registrations made on the parent asset.</span></span>

9. <span data-ttu-id="8496a-131">In het veld **Afwijking onder** voegt u in procenten het laagste niveau voor validatie in als handmatige registraties van activummetingen binnen een verwacht bereik liggen.</span><span class="sxs-lookup"><span data-stu-id="8496a-131">In the **Deviation under** field, insert the lower level in percent for validating if manual asset measure registrations are within an expected range.</span></span> <span data-ttu-id="8496a-132">De validatie is gebaseerd op een lineaire afname van bestaande registraties van activummetingen.</span><span class="sxs-lookup"><span data-stu-id="8496a-132">The validation is based on a linear decrease in existing asset measure registrations.</span></span>
10. <span data-ttu-id="8496a-133">Selecteer in het veld **Type** het type bericht (informatie, waarschuwing, fout) dat moet worden weergegeven als afwijkingen buiten het gedefinieerde bereik optreden bij handmatige registraties van activummetingen.</span><span class="sxs-lookup"><span data-stu-id="8496a-133">In the **Type** field, select the type of message (information, warning, error) to be shown if deviations outside the defined range occur when you make manual asset measure registrations.</span></span>

12. <span data-ttu-id="8496a-135">Voeg op het sneltabblad **Gerelateerde activummetingen** de activummetingen toe die u automatisch wilt bijwerken wanneer deze activummeting wordt bijgewerkt.</span><span class="sxs-lookup"><span data-stu-id="8496a-135">On the **Related asset measures** FastTab, add the asset measures that you want to be automatically updated when this asset measure is updated.</span></span>
