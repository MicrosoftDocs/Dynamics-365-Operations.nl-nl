# <a name="inventory-aging-report-discrepancies"></a>Verschillen in naar ouderdom gerangschikt voorraadrapport

## <a name="symptoms"></a>Symptomen

In het **Naar ouderdom gerangschikt voorraadrapport** worden verschillende waarden weergegeven wanneer ze worden weergegeven in verschillende opslagdimensies (zoals locatie of magazijn).

## <a name="resolution"></a>Oplossing

Zie [Voorbeelden en logica voor naar ouderdom gerangschikt voorraadrapport](/dynamics365/supply-chain/cost-management/inventory-aging-report.md) voor informatie de rapportlogica.
