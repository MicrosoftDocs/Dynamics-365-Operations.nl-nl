# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>Kan een productieorder niet beëindigen vanwege een fout in de waarde consumptionCost stuklijst

## <a name="symptoms"></a>Symptomen

Wanneer u een productieorder probeert te beëindigen, wordt het volgende foutbericht weergegeven:

> Het berekenen van de waarde voor consumptionCost van de stuklijst moet negatief zijn bij uitgifte uit voorraad.

## <a name="resolution"></a>Oplossing

Dit probleem is opgelost in release 10.0.15.
