# <a name="set-up-an-appointment-for-a-load"></a>Een afspraak voor een lading instellen

[!include [banner](../../includes/banner.md)]

Deze procedure laat zien hoe u een dockafspraak voor een lading instelt en plant. Dit wordt gewoonlijk gedaan door een transportco√∂rdinator. U kunt deze procedure gebruiken in het demobedrijf USMF.


## <a name="select-the-load"></a>Selecteer de lading.
1. Ga naar Transportbeheer > Planning > Workbench van ladingplanning.
2. Schakel het selectievakje Verzonden en ontvangen verbergen uit.
3. Selecteer in de lijst de lading met de status Verzonden.
4. Klik op Transport.
5. Klik op Afspraak plannen

## <a name="create-an-appointment"></a>Maak een afspraak
1. Klik op Nieuw.
2. Typ of selecteer een waarde in het veld Afspraakregel.
3. Klik op Opslaan.
4. Klik op Status bijwerken.
5. Klik op Fiatteren.
6. Klik op Opslaan.
7. Sluit de pagina.
