# <a name="set-up-a-transportation-tender"></a>Een transportbetalingsmethode instellen

[!include [banner](../../includes/banner.md)]

Deze procedure laat zien hoe u een Transportaanbesteding instelt. Dit wordt gewoonlijk gedaan door een transportco√∂rdinator. U kunt deze procedure gebruiken in het demobedrijf USMF.


## <a name="select-a-route"></a>Selecteer een route
1. Ga naar Transportbeheer > Planning > Workbench van ladingplanning.
2. Schakel het selectievakje Verzonden en ontvangen verbergen uit.
3. Selecteer de regel met Lading-id 00006.
4. Klik op Classificatie en routering.
5. Klik op Routes.

## <a name="create-the-transportation-tender"></a>Maak de transportaanbesteding
1. Klik op Transportaanbestedingen.
2. Klik op Nieuw.
3. Vouw de sectie Algemeen uit.
4. Typ een getal in het veld Gewenste tarieven.
5. Klik op Opslaan.
6. Klik op Status bijwerken.
7. Klik op Aanbieden.
8. Selecteer een route.
