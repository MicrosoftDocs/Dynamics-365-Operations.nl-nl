# <a name="view-cost-entries-for-a-cost-object"></a>Kostenboekingen voor een kostenobject weergeven

[!include [task guide banner](../../includes/task-guide-banner.md)]

Deze procedure toont hoe u kosteninvoer voor een kostenobject weergeeft. Het bedrijf van de demogegevens dat wordt gebruikt om deze procedure te maken is USMF. Deze procedure is bedoeld voor de kostencontroller.

1. Klik op Kostenadministratie.
2. Klik op Vrijgegeven producten.
3. Gebruik de snelfilter om records te zoeken. Filter bijvoorbeeld op het Veld Artikelnummer met een waarde van 'm0004'.
4. Klik in het actievenster op Kosten beheren.
5. Klik op Kostenobjecten.
6. Klik op Kosteninvoer.
7. Gebruik het snelfilter om op het veld Nummer te filteren met de waarde 'p000031'.
    * Als de kosteninvoer leeg is, stelt u Begindatum in op 31 januari 2012 en Einddatum op 31 december 2012.
