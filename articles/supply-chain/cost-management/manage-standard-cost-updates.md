# <a name="manage-standard-cost-updates"></a><span data-ttu-id="7afed-103">Updates voor standaardkosten beheren</span><span class="sxs-lookup"><span data-stu-id="7afed-103">Manage standard cost updates</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="7afed-104">Updates van standaardkostengegevens kunnen worden beheerd met twee verschillende benaderingen: de benadering met één versie en de benadering met twee versies.</span><span class="sxs-lookup"><span data-stu-id="7afed-104">Updates to standard cost data can be managed by using two different approaches - the one-version approach or the two-version approach.</span></span>

<span data-ttu-id="7afed-105">Bij de benadering met één versie wordt één kostprijsberekeningsversie gebruikt die alle kostenregistraties bevat.</span><span class="sxs-lookup"><span data-stu-id="7afed-105">The one-version approach uses a single costing version that contains all cost records.</span></span> <span data-ttu-id="7afed-106">Deze registraties omvatten de oorspronkelijke koten en alle kostenupdates.</span><span class="sxs-lookup"><span data-stu-id="7afed-106">These records include the original costs and all cost updates.</span></span>

<span data-ttu-id="7afed-107">Bij de methode met twee versies wordt één versie gebruikt die registraties van de oorspronkelijke kosten bevat en een tweede versie die registraties alle van kostenupdates bevat.</span><span class="sxs-lookup"><span data-stu-id="7afed-107">The two-version approach uses one version that contains records of the original costs and a second version that contains records of all cost updates.</span></span> <span data-ttu-id="7afed-108">Het belangrijkste voordeel van de methode met twee versies is de duidelijke afbakening en tracering van kostenupdates in een aparte kostprijsberekeningsversie, zonder dat dit gevolgen heeft voor de oorspronkelijke kostprijsberekeningsversie.</span><span class="sxs-lookup"><span data-stu-id="7afed-108">A primary advantage of the two-version approach is the clear delineation and tracking of cost updates in a separate costing version, without affecting the original costing version.</span></span> <span data-ttu-id="7afed-109">De methode met twee versies kan worden gebruikt om meerdere incrementele updates te identificeren, waarbij elke incrementele update een aparte kostprijsberekeningsversie heeft die de incrementele kostenrecords bevat.</span><span class="sxs-lookup"><span data-stu-id="7afed-109">The two-version approach can be used to identify multiple incremental updates, where each incremental update has a separate costing version that contains the incremental cost records.</span></span>
