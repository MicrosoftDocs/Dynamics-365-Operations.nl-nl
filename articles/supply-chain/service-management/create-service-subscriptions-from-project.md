[!include [banner](../includes/banner.md)]


<span data-ttu-id="59de1-104">U kunt een serviceabonnement maken op basis van een project in **Projectbeheer en boekhouding**.</span><span class="sxs-lookup"><span data-stu-id="59de1-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="59de1-105">Klik op **Projectbeheer en boekhouding** \> **Algemeen** \> **Projecten** \> **Alle projecten**.</span><span class="sxs-lookup"><span data-stu-id="59de1-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="59de1-106">Open het project waarvoor u een abonnement wilt maken.</span><span class="sxs-lookup"><span data-stu-id="59de1-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="59de1-107">Klik in het formulier **Projecten** in het **Actievenster** op het tabblad **Beheren** op **Service** \> **Serviceabonnement**.</span><span class="sxs-lookup"><span data-stu-id="59de1-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

> [!NOTE]
> <P><span data-ttu-id="59de1-112">Als u een nummerreeks voor de abonnementsgroep hebt ingesteld, wordt er automatisch een ID voor het abonnement gegenereerd wanneer u de abonnementsgroep opgeeft.</span><span class="sxs-lookup"><span data-stu-id="59de1-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
