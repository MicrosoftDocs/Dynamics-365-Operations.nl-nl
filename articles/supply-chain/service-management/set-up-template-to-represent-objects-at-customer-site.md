# <a name="set-up-template-to-represent-objects-at-customer-site"></a><span data-ttu-id="7c5e2-103">Sjabloon instellen om objecten op de klantlocatie te vertegenwoordigen</span><span class="sxs-lookup"><span data-stu-id="7c5e2-103">Set up template to represent objects at customer site</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="7c5e2-104">Gebruik deze procedure om een sjaboon in te stellen die objecten op de locatie van een klant vertegenwoordigt.</span><span class="sxs-lookup"><span data-stu-id="7c5e2-104">Use this procedure to set up a template that represents objects at a customer site.</span></span>

1.  <span data-ttu-id="7c5e2-105">Klik op **Servicebeheer** \> **Algemeen** \> **Serviceovereenkomsten** \> **Serviceovereenkomsten**.</span><span class="sxs-lookup"><span data-stu-id="7c5e2-105">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span>

2.  <span data-ttu-id="7c5e2-106">Dubbelklik op een serviceovereenkomst of klik op **Serviceovereenkomst** om een nieuwe serviceovereenkomst te maken.</span><span class="sxs-lookup"><span data-stu-id="7c5e2-106">Double-click a service agreement, or click **Service agreement** to create a new service agreement.</span></span>

3.  <span data-ttu-id="7c5e2-107">Klik op het tabblad **Instellingen**.</span><span class="sxs-lookup"><span data-stu-id="7c5e2-107">Click the **Setup** tab.</span></span>

4.  <span data-ttu-id="7c5e2-108">Klik op **Serviceobjecten** om een sjabloonstuklijst aan de serviceovereenkomst te koppelen.</span><span class="sxs-lookup"><span data-stu-id="7c5e2-108">Click **Service objects** to attach a template bill of materials (BOM) to the service agreement.</span></span>

5.  <span data-ttu-id="7c5e2-109">Selecteer in het formulier **Serviceobjecten** een serviceobject en klik vervolgens op **Functies** \> **Sjabloonstuklijst toevoegen**.</span><span class="sxs-lookup"><span data-stu-id="7c5e2-109">In the **Service objects** form, select a service object, and then click **Functions** \> **Attach template BOM**.</span></span>

6.  <span data-ttu-id="7c5e2-110">Selecteer een sjabloon in het formulier **Sjabloonstuklijst selecteren** in het veld **Sjabloonstuklijst** en klik vervolgens op **OK**.</span><span class="sxs-lookup"><span data-stu-id="7c5e2-110">In the **Select template BOM** form, select a template BOM in the **Template BOM** field, and then click **OK**.</span></span>

7.  <span data-ttu-id="7c5e2-111">Klik in het formulier **Serviceobjecten** op **Ontwerper** om de sjabloonstuklijst zodanig te wijzigen dat hierin de gewenste objecten worden weergegeven.</span><span class="sxs-lookup"><span data-stu-id="7c5e2-111">In the **Service objects** form, click **Designer** to modify the template BOM so that the BOM represents the objects that you want.</span></span>

8.  <span data-ttu-id="7c5e2-112">Sluit de formulieren om de wijzigingen op te slaan.</span><span class="sxs-lookup"><span data-stu-id="7c5e2-112">Close the forms to save your changes.</span></span>
