# <a name="set-up-subscription-groups"></a><span data-ttu-id="2fb19-103">Abonnementsgroepen instellen</span><span class="sxs-lookup"><span data-stu-id="2fb19-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="2fb19-104">Ga naar **Servicebeheer** \> **Instellen** \> **Serviceabonnementen** \> **Abonnementsgroepen**.</span><span class="sxs-lookup"><span data-stu-id="2fb19-104">Go to **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="2fb19-105">Selecteer **Nieuw** om een nieuwe abonnementsgroep te maken.</span><span class="sxs-lookup"><span data-stu-id="2fb19-105">Select **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="2fb19-106">Voer in het veld **Abonnementsgroep** een naam in voor de nieuwe abonnementsgroep.</span><span class="sxs-lookup"><span data-stu-id="2fb19-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="2fb19-107">Selecteer een factuurfrequentie voor het abonnement in het veld **Facturering**.</span><span class="sxs-lookup"><span data-stu-id="2fb19-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="2fb19-108">De factuurfrequentie wordt aangegeven door een periodecode die is ingesteld in het formulier **Periodetypen**.</span><span class="sxs-lookup"><span data-stu-id="2fb19-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="2fb19-109">Als u opbrengst wilt samenvoegen voor abonnementstransacties, schakelt u het selectievakje **Opbrengst samenvoegen** in en selecteert u vervolgens een frequentie in het veld **Toerekening**.</span><span class="sxs-lookup"><span data-stu-id="2fb19-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="2fb19-110">De toerekeningsfrequentie is gebaseerd op een periodecode die is ingesteld in het formulier **Perioden**.</span><span class="sxs-lookup"><span data-stu-id="2fb19-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
