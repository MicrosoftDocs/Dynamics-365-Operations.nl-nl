# <a name="set-up-subscription-groups"></a><span data-ttu-id="bb79d-103">Abonnementsgroepen instellen</span><span class="sxs-lookup"><span data-stu-id="bb79d-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="bb79d-104">Ga naar **Servicebeheer** \> **Instellen** \> **Serviceabonnementen** \> **Abonnementsgroepen**.</span><span class="sxs-lookup"><span data-stu-id="bb79d-104">Go to **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="bb79d-105">Selecteer **Nieuw** om een nieuwe abonnementsgroep te maken.</span><span class="sxs-lookup"><span data-stu-id="bb79d-105">Select **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="bb79d-106">Voer in het veld **Abonnementsgroep** een naam in voor de nieuwe abonnementsgroep.</span><span class="sxs-lookup"><span data-stu-id="bb79d-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="bb79d-107">Selecteer een factuurfrequentie voor het abonnement in het veld **Facturering**.</span><span class="sxs-lookup"><span data-stu-id="bb79d-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="bb79d-108">De factuurfrequentie wordt aangegeven door een periodecode die is ingesteld in het formulier **Periodetypen**.</span><span class="sxs-lookup"><span data-stu-id="bb79d-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>
