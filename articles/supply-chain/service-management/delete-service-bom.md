# <a name="delete-a-service-bom"></a><span data-ttu-id="c3e8e-103">Een servicestuklijst verwijderen</span><span class="sxs-lookup"><span data-stu-id="c3e8e-103">Delete a service BOM</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="c3e8e-104">Klik op **Servicebeheer** \> **Algemeen** \> **Serviceovereenkomsten** \> **Serviceovereenkomsten**.</span><span class="sxs-lookup"><span data-stu-id="c3e8e-104">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span> <span data-ttu-id="c3e8e-105">Selecteer de serviceovereenkomst waar u een servicestuklijst van wilt verwijderen.</span><span class="sxs-lookup"><span data-stu-id="c3e8e-105">Select the service agreement that you want to delete a service bill of materials (BOM) from.</span></span>

2.  <span data-ttu-id="c3e8e-106">Klik in het **Actievenster** op het tabblad **Serviceobjecten**.</span><span class="sxs-lookup"><span data-stu-id="c3e8e-106">On the **Action Pane**, click **Service objects**.</span></span>

3.  <span data-ttu-id="c3e8e-107">Selecteer in het formulier **Serviceobjecten** het serviceobject waaruit u een servicestuklijst wilt verwijderen.</span><span class="sxs-lookup"><span data-stu-id="c3e8e-107">In the **Service objects** form, select the service object that you want to delete a service BOM from.</span></span> <span data-ttu-id="c3e8e-108">Klik op **Functies** en selecteer vervolgens **Servicestuklijst verwijderen**.</span><span class="sxs-lookup"><span data-stu-id="c3e8e-108">Click **Functions**, and then select **Delete service BOM**.</span></span>

4.  <span data-ttu-id="c3e8e-109">Klik op **Ja**.</span><span class="sxs-lookup"><span data-stu-id="c3e8e-109">Click **Yes**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="c3e8e-110">Als u geen sjabloonstuklijst hebt toegepast, kan de servicestuklijst worden verwijderd uit het formulier <STRONG>Sjabloonstuklijsten</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="c3e8e-110">If you have not applied a template BOM, the service BOM can be deleted from the <STRONG>Template BOMs</STRONG> form.</span></span> <span data-ttu-id="c3e8e-111">Hebt u een stuklijstsjabloon echter toegepast op een servicestuklijst, dan kunt u deze niet met deze methode verwijderen.</span><span class="sxs-lookup"><span data-stu-id="c3e8e-111">If you have applied a template BOM, the service BOM cannot be deleted by using this method.</span></span></P>
