# <a name="create-subscription-fee-transactions"></a>Tarieftransacties abonnement maken 

[!include [banner](../includes/banner.md)]


1.  Klik op **Servicebeheer** \> **Algemeen** \> **Serviceabonnementen** \> **Alle serviceabonnementen**.

2.  Selecteer op de lijstpagina **Alle serviceabonnementen** het abonnement waarvoor u een kostentransactie voor abonnementen wilt maken.

3.  Klik in het **Actievenster** op het tabblad **Abonnement** op **Abonnementskosten**.

4.  Voer in het formulier **Abonnementskosten maken** de juiste informatie in de velden in.

5.  Klik tot slot op **OK**.
