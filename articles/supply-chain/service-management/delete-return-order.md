# <a name="delete-a-return-order"></a><span data-ttu-id="93598-103">Een retourorder verwijderen</span><span class="sxs-lookup"><span data-stu-id="93598-103">Delete a return order</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="93598-104">Met het formulier **Retourorder verwijderen** kunt u meerdere retourorders verwijderen.</span><span class="sxs-lookup"><span data-stu-id="93598-104">Use the **Delete return order** form to delete multiple return orders.</span></span>

1.  <span data-ttu-id="93598-105">Klik op **Verkoop en marketing** \> **Periodiek** \> **Opschonen** \> **Retourorders verwijderen**.</span><span class="sxs-lookup"><span data-stu-id="93598-105">Click **Sales and marketing** \> **Periodic** \> **Clean up** \> **Delete return orders**.</span></span>

2.  <span data-ttu-id="93598-106">Klik op **Selecteren** om het formulier **Bijwerking van verkoop** te openen.</span><span class="sxs-lookup"><span data-stu-id="93598-106">Click **Select** to open the **Sales update** form.</span></span>

4.  <span data-ttu-id="93598-109">Selecteer de regel van de retourorder die moet worden verwijderd en klik op **OK**.</span><span class="sxs-lookup"><span data-stu-id="93598-109">Select the return order line to delete, and then click **OK**.</span></span>

5.  <span data-ttu-id="93598-110">Klik op **Ja** om de retourorders te verwijderen.</span><span class="sxs-lookup"><span data-stu-id="93598-110">Click **Yes** to delete the return orders.</span></span>
