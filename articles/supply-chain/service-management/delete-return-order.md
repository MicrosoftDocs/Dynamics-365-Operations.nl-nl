[!include [banner](../includes/banner.md)]


<span data-ttu-id="dd9a1-104">Met het formulier **Retourorder verwijderen** kunt u meerdere retourorders verwijderen.</span><span class="sxs-lookup"><span data-stu-id="dd9a1-104">Use the **Delete return order** form to delete multiple return orders.</span></span>

1.  <span data-ttu-id="dd9a1-105">Klik op **Verkoop en marketing** \> **Periodiek** \> **Opschonen** \> **Retourorders verwijderen**.</span><span class="sxs-lookup"><span data-stu-id="dd9a1-105">Click **Sales and marketing** \> **Periodic** \> **Clean up** \> **Delete return orders**.</span></span>
