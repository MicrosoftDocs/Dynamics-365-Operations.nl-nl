# <a name="set-up-service-intervals"></a><span data-ttu-id="05344-103">Service-intervallen instellen</span><span class="sxs-lookup"><span data-stu-id="05344-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="05344-104">Met service-intervallen wordt aangegeven hoe vaak er serviceorderregels voor serviceovereenkomstregels worden gemaakt wanneer u serviceorders maakt.</span><span class="sxs-lookup"><span data-stu-id="05344-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="05344-105">Klik op **Servicebeheer** \> **Instellen** \> **Serviceovereenkomsten** \> **Service-intervallen**.</span><span class="sxs-lookup"><span data-stu-id="05344-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="05344-106">Maak een nieuw service-interval.</span><span class="sxs-lookup"><span data-stu-id="05344-106">Create a new service interval.</span></span>
3. <span data-ttu-id="05344-107">Geef de id en de omschrijving van het service-interval op.</span><span class="sxs-lookup"><span data-stu-id="05344-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="05344-108">Selecteer het bereik in het veld **Bereik**.</span><span class="sxs-lookup"><span data-stu-id="05344-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="05344-109">Voer de frequentie in het veld **Frequentie** in.</span><span class="sxs-lookup"><span data-stu-id="05344-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="05344-110">De frequentie is de factor waarmee u het bereik moet vermenigvuldigen om het interval voor een serviceovereenkomst te verkrijgen.</span><span class="sxs-lookup"><span data-stu-id="05344-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="05344-111">Druk op **Alt+S** om het service-interval op te slaan.</span><span class="sxs-lookup"><span data-stu-id="05344-111">Press **Alt+S** to save the service interval.</span></span>
