[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="ff16c-104">In dit onderwerp wordt uitgelegd hoe u een locatieprofiel maakt in Dynamics 365 for Finance and Operations.</span><span class="sxs-lookup"><span data-stu-id="ff16c-104">This topic explains how to create a location profile in Dynamics 365 for Finance and Operations.</span></span> <span data-ttu-id="ff16c-105">Elke locatie in het magazijn moet een locatieprofiel hebben dat eraan is gekoppeld en dat de eigenschappen beschrijft van de locatie, bijvoorbeeld of de locatie gemengde artikelen toestaat.</span><span class="sxs-lookup"><span data-stu-id="ff16c-105">Every location in the warehouse needs to have a location profile associated with it that describes the properties of the location, for example, whether the location allows mixed items.</span></span> <span data-ttu-id="ff16c-106">In deze procedure wordt een profiel gemaakt voor een locatie die geen controlenummerplaat vereist.</span><span class="sxs-lookup"><span data-stu-id="ff16c-106">In this procedure we’ll create a profile for a location that doesn’t require license plate control.</span></span> <span data-ttu-id="ff16c-107">Gemengde artikelen en gemengde voorraadstatussen worden ingeschakeld en cyclustelling is toegestaan.</span><span class="sxs-lookup"><span data-stu-id="ff16c-107">We’ll enable mixed items, and mixed inventory statuses, and allow cycle counting.</span></span> <span data-ttu-id="ff16c-108">U kunt deze procedure gebruiken in het demobedrijf USMF.</span><span class="sxs-lookup"><span data-stu-id="ff16c-108">You can use this procedure in the USMF demo data company.</span></span>


1. <span data-ttu-id="ff16c-109">Ga in het navigatievenster naar **Modules > Magazijnbeheer > Instellingen > Magazijn > Locatieprofielen**.</span><span class="sxs-lookup"><span data-stu-id="ff16c-109">In the navigation pane, go to **Modules > Warehouse management > Setup > Warehouse > Location profiles**.</span></span>
2. <span data-ttu-id="ff16c-110">Selecteer **Nieuw**.</span><span class="sxs-lookup"><span data-stu-id="ff16c-110">Select **New**.</span></span>
3. <span data-ttu-id="ff16c-111">Typ een waarde in het veld **Locatieprofiel-id**.</span><span class="sxs-lookup"><span data-stu-id="ff16c-111">In the **Location profile ID** field, type a value.</span></span>
4. <span data-ttu-id="ff16c-112">Typ een waarde in het veld **Naam**.</span><span class="sxs-lookup"><span data-stu-id="ff16c-112">In the **Name** field, type a value.</span></span>
5. <span data-ttu-id="ff16c-113">Typ of selecteer een waarde in het veld **Locatie-indeling**.</span><span class="sxs-lookup"><span data-stu-id="ff16c-113">In the **Location format** field, enter or select a value.</span></span>
6. <span data-ttu-id="ff16c-114">Typ of selecteer een waarde in het veld **Locatietype**.</span><span class="sxs-lookup"><span data-stu-id="ff16c-114">In the **Location type** field, enter or select a value.</span></span>
7. <span data-ttu-id="ff16c-115">Typ of selecteer een waarde in het veld **Id dockbeheerprofiel**.</span><span class="sxs-lookup"><span data-stu-id="ff16c-115">In the **Dock management profile ID** field, enter or select a value.</span></span>
8. <span data-ttu-id="ff16c-116">Selecteer **Ja** in het veld **Gemengde artikelen toestaan**.</span><span class="sxs-lookup"><span data-stu-id="ff16c-116">Select **Yes** in the **Allow mixed items** field.</span></span>
9. <span data-ttu-id="ff16c-117">Selecteer **Ja** in het veld **Gemengde voorraadstatussen toestaan**.</span><span class="sxs-lookup"><span data-stu-id="ff16c-117">Select **Yes** in the **Allow mixed inventory statuses** field.</span></span>
10. <span data-ttu-id="ff16c-118">Selecteer **Ja** in het veld **Cyclustelling toestaan**.</span><span class="sxs-lookup"><span data-stu-id="ff16c-118">Select **Yes** in the **Allow cycle counting** field.</span></span>
11. <span data-ttu-id="ff16c-119">Selecteer **Opslaan**.</span><span class="sxs-lookup"><span data-stu-id="ff16c-119">Select **Save**.</span></span>
