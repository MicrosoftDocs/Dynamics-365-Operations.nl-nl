# <a name="create-a-formula-by-copying-from-an-existing-formula"></a>Een formule maken door uit een bestaande formule te kopiëren

[!include [banner](../../includes/banner.md)]

Deze procedure toont hoe u snel een nieuwe formule maakt door uit een bestaande formule te kopiëren. U kunt de demobedrijven USMF, USP2 of USPI gebruiken om deze procedure te maken.


## <a name="create-a-new-formula"></a>Een nieuwe formule maken
1. Ga naar Formules.
2. Klik op Nieuw.
3. Typ een waarde in het veld Formule.
4. Typ een waarde in het veld Naam.
5. Typ of selecteer een waarde in het veld Locatie.
6. Typ of selecteer een waarde in het veld Artikelgroep.
7. Klik op Opslaan.

## <a name="copy-from-an-existing-formula"></a>Uit een bestaande formule kopiëren
1. Klik op Kopiëren.
2. Typ of selecteer een waarde in het veld Artikelnummer.
3. Typ of selecteer een waarde in het veld Formuleversie.
4. Klik op OK.

## <a name="revise-the-copied-formula"></a>De gekopieerde formule controleren
1. Zoek en selecteer de gewenste record in de lijst.
2. Klik op Verwijderen.
3. Klik op Ja.

## <a name="approve-formula"></a>Formule goedkeuren
1. Klik op Formule goedkeuren.
2. Typ of selecteer een waarde in het veld Goedgekeurd door.
3. Klik op OK.
4. Sluit de pagina.
