[!include [banner](../../includes/banner.md)]

<span data-ttu-id="c7a62-104">In dit onderwerp wordt voorgedaan hoe u een recordsjabloon kunt maken, zodat vaak gebruikte veldwaarden niet expliciet moeten worden ingevoerd voor elke nieuwe record.</span><span class="sxs-lookup"><span data-stu-id="c7a62-104">This topic demonstrates how to create a record template so that field values that are used often do not have to be entered explicitly for each new record.</span></span> <span data-ttu-id="c7a62-105">In deze procedure maakt u een nieuwe record voor nieuwe laptops die u wilt laten toevoegen aan uw vaste activa.</span><span class="sxs-lookup"><span data-stu-id="c7a62-105">In this procedure, you'll create a new record for new laptops that should be added to your fixed assets.</span></span> <span data-ttu-id="c7a62-106">In deze procedure wordt het voorbeeldbedrijf USMF gebruikt.</span><span class="sxs-lookup"><span data-stu-id="c7a62-106">This procedure uses the USMF sample company.</span></span>

1. <span data-ttu-id="c7a62-107">Ga in het navigatievenster naar **Modules > Vaste activa > Vaste activa > Vaste activa**.</span><span class="sxs-lookup"><span data-stu-id="c7a62-107">In the navigation pane, go to **Modules > Fixed assets > Fixed assets > Fixed assets**.</span></span>
2. <span data-ttu-id="c7a62-108">Selecteer **Nieuw**.</span><span class="sxs-lookup"><span data-stu-id="c7a62-108">Select **New**.</span></span>
3. <span data-ttu-id="c7a62-109">Typ of selecteer een waarde in het veld **Groep vaste activa**.</span><span class="sxs-lookup"><span data-stu-id="c7a62-109">In the **Fixed asset group** field, enter or select a value.</span></span>
4. <span data-ttu-id="c7a62-110">Typ een waarde in het veld **Naam**.</span><span class="sxs-lookup"><span data-stu-id="c7a62-110">In the **Name** field, type a value.</span></span> <span data-ttu-id="c7a62-111">Voer bijvoorbeeld **Corporate lead-laptop** in.</span><span class="sxs-lookup"><span data-stu-id="c7a62-111">For example, enter **Corporate lead laptop**.</span></span>  
5. <span data-ttu-id="c7a62-112">Typ een waarde in het veld **Zoeknaam**.</span><span class="sxs-lookup"><span data-stu-id="c7a62-112">In the **Search name** field, type a value.</span></span> <span data-ttu-id="c7a62-113">Voer bijvoorbeeld **laptop** in.</span><span class="sxs-lookup"><span data-stu-id="c7a62-113">For example, enter **laptop**.</span></span>

9. <span data-ttu-id="c7a62-117">Selecteer **Recordinfo**.</span><span class="sxs-lookup"><span data-stu-id="c7a62-117">Select **Record info**.</span></span>
10. <span data-ttu-id="c7a62-118">Selecteer **Gebruikerssjabloon**.</span><span class="sxs-lookup"><span data-stu-id="c7a62-118">Select **User template**.</span></span>
11. <span data-ttu-id="c7a62-119">Typ een waarde in het veld **Naam**.</span><span class="sxs-lookup"><span data-stu-id="c7a62-119">In the **Name** field, type a value.</span></span>
12. <span data-ttu-id="c7a62-120">Typ een waarde in het veld **Beschrijving**.</span><span class="sxs-lookup"><span data-stu-id="c7a62-120">In the **Description** field, type a value.</span></span>
13. <span data-ttu-id="c7a62-121">Selecteer **OK**.</span><span class="sxs-lookup"><span data-stu-id="c7a62-121">Select **OK**.</span></span>
