# <a name="upper-er-function"></a>De ER-functie UPPER

[!include [banner](../includes/banner.md)]

De functie `UPPER` retourneert de opgegeven tekstreeks als een waarde van het type *Tekenreeks* nadat deze is geconverteerd naar hoofdletters.

## <a name="syntax"></a>Syntaxis

```vb
UPPER (text )
```

## <a name="arguments"></a>Argumenten

`text`: *Tekenreeks*

Het geldige pad van een gegevensbron van het type *Tekenreeks*.

## <a name="return-values"></a>Retourwaarden

*Tekenreeks*

De resulterende tekstwaarde.

## <a name="example"></a>Voorbeeld

`UPPER ("Sample")` retourneert **SAMPLE**.
