# <a name="list-of-er-functions-in-the-mathematical-category"></a>Lijst met ER-functies in de wiskundige categorie

[!include [banner](../includes/banner.md)]

Wiskundige functies voor elektronische rapportage (ER) kunnen worden gebruikt om een groot aantal wiskundige berekeningen uit te voeren. In dit onderwerp vindt u een overzicht van deze functies.

## <a name="list-of-supported-functions"></a>Lijst met ondersteunde functies

| Functie | Beschrijving |
|----------|-------------|
| [Abs](er-functions-mathematical-abs.md)             | Deze functie retourneert de absolute waarde (modulus) van de opgegeven numerieke waarde als een *werkelijke* waarde. Met andere woorden, het getal wordt zonder teken geretourneerd. |
| [Vermogen](er-functions-mathematical-power.md)         | Deze functie retourneert een *werkelijke* waarde die het resultaat vertegenwoordigt van het verheffen van het opgegeven positieve getal naar de opgegeven macht. |
| [Round](er-functions-mathematical-round.md)         | Deze functie retourneert het opgegeven getal als een *werkelijke* waarde nadat deze is afgerond op het opgegeven aantal decimalen. |
| [RoundDown](er-functions-mathematical-rounddown.md) | Deze functie retourneert het opgegeven getal als een *werkelijke* waarde nadat deze naar beneden is afgerond op het opgegeven aantal decimalen. |
| [RoundUp](er-functions-mathematical-roundup.md)     | Deze functie retourneert het opgegeven getal als een *werkelijke* waarde nadat deze naar boven is afgerond op het opgegeven aantal decimalen. |
