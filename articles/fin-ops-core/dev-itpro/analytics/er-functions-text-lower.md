# <a name="LOWER">De ER-functie LOWER</a>

[!include [banner](../includes/banner.md)]

De functie `LOWER` retourneert de opgegeven tekstreeks als een waarde van het type *Tekenreeks* nadat deze is geconverteerd naar kleine letters.

## <a name="syntax"></a>Syntaxis

```
LOWER (text)
```

## <a name="arguments"></a>Argumenten

`text`: *Tekenreeks*

Een *tekenreekswaarde* waarmee de tekst wordt opgegeven.

## <a name="return-values"></a>Retourwaarden

*Tekenreeks*

De resulterende tekstwaarde.

## <a name="example"></a>Voorbeeld

`LOWER ("Sample")` retourneert **sample**.
