# <a name="file-destination"></a>Bestandsbestemming

[!include [banner](../includes/banner.md)]

U kunt een bestandsbestemming configureren voor elke MAP- of BESTAND-component van een ER-indeling (elektronische rapportage) die wordt geconfigureerd voor het genereren van uitgaande documenten. Op basis van de instelling van de bestemming is een gegenereerd document beschikbaar voor downloaden vanuit de webbrowser.

Als u op de pagina **Bestemmingsinstellingen** **Ingeschakeld** instelt op **Ja**, wordt een dialoogvenster voor openen of opslaan weergegeven wanneer de uitvoering van de configuratie is voltooid.
