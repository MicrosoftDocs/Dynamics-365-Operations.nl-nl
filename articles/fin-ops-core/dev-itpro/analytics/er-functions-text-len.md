# <a name="len-er-function"></a>De ER-functie LEN

[!include [banner](../includes/banner.md)]

De functie `LEN` retourneert een waarde van het type *Geheel getal* voor het aantal tekens in de opgegeven tekenreeks.

## <a name="syntax"></a>Syntaxis

```vb
LEN (text)
```

## <a name="arguments"></a>Argumenten

`text`: *Tekenreeks*

Een *tekenreekswaarde* waarmee de tekst wordt opgegeven.

## <a name="return-values"></a>Retourwaarden

*Geheel getal*

De resulterende numerieke waarde.

## <a name="example"></a>Voorbeeld

`LEN ("Sample")` retourneert **6**.
