# <a name="supported-web-browsers-for-lifecycle-services"></a>Ondersteunde webbrowsers voor Lifecycle Services 

[!include[banner](../includes/banner.md)]

Gebruikers hebben toegang tot Microsoft Dynamics Lifecyle Services (LCS) via de meest recente versies van deze populair browsers: 

- Microsoft Edge (aanbevolen: [Edge op basis van Chromium](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Google Chrome
- Apple Safari

Als u de nieuwste versie van elke webbrowser wilt opzoeken, gaat u naar de website van de softwarefabrikant. Voor optimale prestaties en een optimale ervaring raden we u aan om de meest recente versie van een moderne browser te gebruiken, met name Microsoft Edge.
