    - <span data-ttu-id="c26be-120">De nummerreeksen van alle bereiken kunnen *constante segmenten* en *alfanumerieke segmenten* bevatten.</span><span class="sxs-lookup"><span <span data-ttu-id="c26be-122">Gebruik dit segmenttype om een koppelteken of andere scheidingstekens toe te voegen tussen nummerreekssegmenten.</span><span class="sxs-lookup"><span
