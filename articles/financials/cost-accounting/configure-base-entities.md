# <a name="configure-base-entities"></a>Basisentiteiten configureren 

[!include[banner](../includes/banner.md)]


Kostprijsboekhouding is een onafhankelijke module waarvoor de volgende basisentiteiten correct moeten worden geconfigureerd in Dynamics 365 for Finance and Operations, Enterprise edition.

-  Valuta
-  Wisselkoers
-  Fiscale kalender  

Voordat u de module Kostprijsboekhouding kunt gebruiken, moet u een valuta, wisselkoers en fiscale kalender hebben gedefinieerd voor uw bedrijf. 

Voor meer informatie over deze entiteiten volgt u de eLearning-trainingscursus [Module 2 Wisselkoers en wisselkoers instellen](https://mbspartner.microsoft.com/AX/CourseModules/1215) (hiervoor hebt u een CustomerSource-account nodig).
