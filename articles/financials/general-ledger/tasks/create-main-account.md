3. <span data-ttu-id="f86fa-108">Typ een waarde in het veld Hoofdrekening.</span><span class="sxs-lookup"><span data-stu-id="f86fa-108">In the Main account field, type a value.</span></span>

    * <span data-ttu-id="f86fa-114">Wijzig het standaard debet- of creditsaldo.</span><span class="sxs-lookup"><span data-stu-id="f86fa-114">Change the default debit or credit balance.</span></span>

9. <span data-ttu-id="f86fa-116">Zoek en selecteer het gewenste record in de lijst.</span><span class="sxs-lookup"><span data-stu-id="f86fa-116">In the list, find and select the desired record.</span></span>
