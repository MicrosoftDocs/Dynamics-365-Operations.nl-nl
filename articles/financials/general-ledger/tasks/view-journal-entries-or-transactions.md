# <a name="view-journal-entries-or-transactions"></a><span data-ttu-id="97a83-103">Journaalboekingen of transacties weergeven</span><span class="sxs-lookup"><span data-stu-id="97a83-103">View journal entries or transactions</span></span>

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="97a83-104">In deze procedure ziet u hoe u met het Boekstuktransactiesonderzoek kunt zoeken naar journaalboekingen of transacties.</span><span class="sxs-lookup"><span data-stu-id="97a83-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="97a83-105">Ga naar Grootboek > Query's en rapporten > Boekstuktransacties.</span><span class="sxs-lookup"><span data-stu-id="97a83-105">Go to General ledger > Inquiries and reports > Voucher transactions.</span></span>
2. <span data-ttu-id="97a83-106">Selecteer het veld waarvoor u filtercriteria wilt definiëren.</span><span class="sxs-lookup"><span data-stu-id="97a83-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="97a83-107">Voer de filtercritieria voor het geselecteerde veld in.</span><span class="sxs-lookup"><span data-stu-id="97a83-107">Enter your filter criteria for the selected field.</span></span>
    * <span data-ttu-id="97a83-108">U kunt flteren op een enkele waarde of een bereik.</span><span class="sxs-lookup"><span data-stu-id="97a83-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="97a83-109">Let er bij het definiëren van een bereik op dat de correcte syntaxis wordt gebruikt.</span><span class="sxs-lookup"><span data-stu-id="97a83-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="97a83-110">Scheid de waarden door middel van twee punten (..).</span><span class="sxs-lookup"><span data-stu-id="97a83-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="97a83-111">Klik op het tabblad Joins om extra tabellen toe te voegen waaruit u wilt filteren.</span><span class="sxs-lookup"><span data-stu-id="97a83-111">Click the Joins tab to add additional tables from which to filter.</span></span>
5. <span data-ttu-id="97a83-112">Selecteer in de structuur 'Tabellen\Algemene journaalpost'.</span><span class="sxs-lookup"><span data-stu-id="97a83-112">In the tree, select 'Tables\General journal entry'.</span></span>
6. <span data-ttu-id="97a83-113">Klik op Tabeljoin toevoegen.</span><span class="sxs-lookup"><span data-stu-id="97a83-113">Click Add table join.</span></span>
7. <span data-ttu-id="97a83-114">Klik op Annuleren als u geen extra tabel wilt toevoegen.</span><span class="sxs-lookup"><span data-stu-id="97a83-114">Click Cancel if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="97a83-115">Klik op het tabblad Bereik.</span><span class="sxs-lookup"><span data-stu-id="97a83-115">Click the Range tab.</span></span>
9. <span data-ttu-id="97a83-116">Klik op OK om de query uit te voeren.</span><span class="sxs-lookup"><span data-stu-id="97a83-116">Click OK to run the query.</span></span>
10. <span data-ttu-id="97a83-117">Klik op Oorsprong van transactie.</span><span class="sxs-lookup"><span data-stu-id="97a83-117">Click Transaction origin.</span></span>
    * <span data-ttu-id="97a83-118">Met verschillende knoppen op het raster kunt u aanvullende informatie onderzoeken over de geselecteerde record van het boekstuk.</span><span class="sxs-lookup"><span data-stu-id="97a83-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="97a83-119">Sommige knoppen zijn mogelijk niet beschikbaar, afhankelijk van het type transactie en de kenmerken van de transactie.</span><span class="sxs-lookup"><span data-stu-id="97a83-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>  
11. <span data-ttu-id="97a83-120">Sluit de pagina.</span><span class="sxs-lookup"><span data-stu-id="97a83-120">Close the page.</span></span>
12. <span data-ttu-id="97a83-121">Klik op Oorspronkelijk document.</span><span class="sxs-lookup"><span data-stu-id="97a83-121">Click Original document.</span></span>
13. <span data-ttu-id="97a83-122">Sluit de pagina.</span><span class="sxs-lookup"><span data-stu-id="97a83-122">Close the page.</span></span>
