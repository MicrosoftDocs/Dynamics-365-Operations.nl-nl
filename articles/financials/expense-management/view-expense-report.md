# <a name="view-an-expense-report"></a><span data-ttu-id="eb4ca-103">Een onkostennota weergeven</span><span class="sxs-lookup"><span data-stu-id="eb4ca-103">View an expense report</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="eb4ca-104">Wanneer u een query uitvoert om transacties te bekijken in het Grootboek of Inkoop en sourcing, geeft de query mogelijk te veel informatie.</span><span class="sxs-lookup"><span data-stu-id="eb4ca-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="eb4ca-105">U wilt daarom uw zoekopdracht aanpassen.</span><span class="sxs-lookup"><span data-stu-id="eb4ca-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="eb4ca-106">Voor de informatie die u nodig hebt, kunt u op een transactie inzoomen om de bijbehorende onkostennota te bekijken.</span><span class="sxs-lookup"><span data-stu-id="eb4ca-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="eb4ca-107">Voor onkostennota's kunt u vanuit een transactie inzoomen op de oorspronkelijke nota die is opgeslagen in Onkostenbeheer.</span><span class="sxs-lookup"><span data-stu-id="eb4ca-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="eb4ca-108">Naast het totale bedrag aan onkosten kunt u de gespecificeerde transacties, de transacties die in aanmerking komen voor btw-teruggave en de ontvangstbewijzen weergeven die zijn gekoppeld aan de onkosten.</span><span class="sxs-lookup"><span data-stu-id="eb4ca-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="eb4ca-109">Als u wilt inzoomen naar de oorspronkelijke onkostennota met de transactie, selecteert u de transactie en vervolgens het **oorspronkelijke document**.</span><span class="sxs-lookup"><span data-stu-id="eb4ca-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="eb4ca-110">Als u afzonderlijke transacties weergeeft, kunt u ook alle aanvullende gegevens weergeven die aan een transactie zijn gekoppeld.</span><span class="sxs-lookup"><span data-stu-id="eb4ca-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="eb4ca-111">U kunt bijvoorbeeld voor de gespecificeerde onkosten voor een hotel een gedetailleerde lijst met onkosten weergeven, zoals internetgebruik, telefoongesprekken en roomservice.</span><span class="sxs-lookup"><span data-stu-id="eb4ca-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
