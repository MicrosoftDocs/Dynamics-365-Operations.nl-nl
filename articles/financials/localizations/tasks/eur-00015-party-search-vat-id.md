<span data-ttu-id="552d7-106">Deze procedure is van toepassing op alle Europese landen/regio's.</span><span class="sxs-lookup"><span data-stu-id="552d7-106">This procedure applies to all European countries/regions.</span></span> <span data-ttu-id="552d7-107">De procedure is gemaakt met het demobedrijf DEMF met een primair adres in Duitsland.</span><span class="sxs-lookup"><span data-stu-id="552d7-107">The procedure was created using the demo data company DEMF with a primary address in Germany.</span></span> <span data-ttu-id="552d7-108">Deze procedure is bedoeld voor een beheerder van leveranciers of klanten.</span><span class="sxs-lookup"><span data-stu-id="552d7-108">This procedure is intended for an accounts payable manager or accounts receivable manager.</span></span> <span data-ttu-id="552d7-109">Deze procedure is voor een functie die in versie 1611 van Dynamics 365 for Operations is toegevoegd.</span><span class="sxs-lookup"><span data-stu-id="552d7-109">This procedure is for a feature that was added in Dynamics 365 for Operations version 1611.</span></span>
