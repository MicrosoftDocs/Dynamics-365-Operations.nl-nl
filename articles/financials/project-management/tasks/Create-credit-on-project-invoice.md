# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="ef3aa-103">Een creditnota voor projectfacturen maken</span><span class="sxs-lookup"><span data-stu-id="ef3aa-103">Create a credit note on project invoices</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

1. <span data-ttu-id="ef3aa-104">Ga naar Projectbeheer en boekhouding > Projecten > Alle projecten.</span><span class="sxs-lookup"><span data-stu-id="ef3aa-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="ef3aa-105">Zoek en selecteer de gewenste record in de lijst.</span><span class="sxs-lookup"><span data-stu-id="ef3aa-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="ef3aa-106">Selecteer de project-id die wordt gebruikt voor de creditnota.</span><span class="sxs-lookup"><span data-stu-id="ef3aa-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="ef3aa-107">Klik in de lijst op de koppeling in de geselecteerde rij.</span><span class="sxs-lookup"><span data-stu-id="ef3aa-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="ef3aa-108">Klik in het actievenster op Beheren.</span><span class="sxs-lookup"><span data-stu-id="ef3aa-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="ef3aa-109">Klik op Factuurjournalen.</span><span class="sxs-lookup"><span data-stu-id="ef3aa-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="ef3aa-110">Selecteer de transactie voor de creditnota.</span><span class="sxs-lookup"><span data-stu-id="ef3aa-110">Select the transaction for credit note.</span></span> <span data-ttu-id="ef3aa-111">Markeer in de lijst de geselecteerde rij.</span><span class="sxs-lookup"><span data-stu-id="ef3aa-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="ef3aa-112">Klik op Functies.</span><span class="sxs-lookup"><span data-stu-id="ef3aa-112">Click Functions.</span></span> 
8. <span data-ttu-id="ef3aa-113">Klik op Selecteren voor creditnota.</span><span class="sxs-lookup"><span data-stu-id="ef3aa-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="ef3aa-114">Schakel het selectievakje in om de transactie te selecteren waarvoor u een creditnota wilt maken.</span><span class="sxs-lookup"><span data-stu-id="ef3aa-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="ef3aa-115">Klik op OK.</span><span class="sxs-lookup"><span data-stu-id="ef3aa-115">Click OK.</span></span> 

<span data-ttu-id="ef3aa-116">Na deze stap is de creditnota beschikbaar voor factuurvoorstellen.</span><span class="sxs-lookup"><span data-stu-id="ef3aa-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
