# <a name="project-mobile-applications-overview"></a>Projectoverzicht mobiele toepassingen

[!include [banner](../includes/banner.md)]

Drie mobiele toepassingen die zijn gerelateerd aan projecttijd zijn beschikbaar voor Microsoft Dynamics 365: Microsoft Dynamics 365 Project Timesheet, Projecttijdinvoer en Mijn urenstaten/Urenstaten voor mijn controle (geoptimaliseerd voor mobiel).

## <a name="project-timesheet-mobile-app"></a>Mobiele app voor projecturenstaat

Met de mobiele app Dynamics 365 Project Timesheet kunnen gebruikers urenstaten indienen en goedkeuren voor projecten op hun mobiele apparaat. Deze mobiele app bevat de urenstaatfunctionaliteit in het gebied Projectbeheer en boekhouding van Microsoft Dynamics 365 for Finance and Operations. Met deze app worden de gebruikersproductiviteit en efficiÃ«ntie verbeterd en wordt ook de tijdige invoer en goedkeuring van projecturenstaten mogelijk gemaakt.

## <a name="project-time-entry-workspace"></a>Werkgebied Projecttijdinvoer

Met het werkgebied Projecttijdinvoer kunnen gebruikers tijd invoeren en opslaan voor een project. In dit werkgebied kunnen gebruikers echter geen projecturenstaten indienen of goedkeuren. Het kan alleen worden gebruikt voor tijdinvoer. Dit mobiele werkgebied is bedoeld om samen te worden gebruikt met de toepassing Microsoft Dynamics 365 for Unified Operations Mobile.

## <a name="my-timesheetstimesheets-for-my-review"></a>Mijn urenstaten/Urenstaten voor mijn controle

Mijn urenstaten (geoptimaliseerd voor mobiel) en Urenstaten voor mijn controle (geoptimaliseerd voor mobiel) vormen een oplossing waarmee werknemers projecturenstaten op een mobiel apparaat kunnen maken, indienen en goedkeuren. Werknemers kunnen toegang krijgen tot de functionaliteit Finance and Operations via de URL voor de toepassing. Voor deze oplossing is de toepassing Dynamics 365 for Unified Operations Mobile niet vereist.
