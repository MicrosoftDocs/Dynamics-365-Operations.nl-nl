# <a name="completion-status"></a>Status van voltooiing


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In dit onderwerp wordt de optieset voor Status van voltooiing voor Dynamics 365 Human Resources beschreven.

Fysieke naam: mshr_hcmcompletionstatus

Voor deze opsomming wordt de optieset met statuswaarden ingesteld voor de screenings voor de kandidaat. 

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | Niet voltooid | De kandidaat heeft de screening nog niet voltooid. |
| 200000001 | Goedgekeurd | De kandidaat is geslaagd voor de screening. |
| 200000002 | Afgekeurd | De kandidaat is niet geslaagd voor de screening. |
