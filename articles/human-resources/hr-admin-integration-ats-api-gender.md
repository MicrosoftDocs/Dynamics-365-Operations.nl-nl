# <a name="gender"></a>Geslacht

In dit onderwerp wordt de optieset voor Gender voor Dynamics 365 Human Resources beschreven.

Fysieke naam: mshr_hcmpersongender

Voor deze opsomming wordt de optieset genders ingesteld voor de kandidaat. Dit is beschikbaar in de optieset mshr_hcmpersongender.

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | None | Er is geen gender opgegeven. |
| 200000001 | Mannelijk | Man. |
| 200000002 | Vrouwelijk | Vrouw. |
| 200000003 | NonSpecific | Selectie voor een niet-specifiek gender. |
