# <a name="regulatory-job-category"></a>Categorie wettelijk voorgeschreven taken


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In dit artikel wordt de optieset voor Wettelijke taakcategorie voor Dynamics 365 Human Resources beschreven.

Fysieke naam: mshr_hcmregulatoryjobcategory

Deze opsomming specificeert de optieset voor de waarden van de optieset voor wettelijke taakcategorie. Dit wordt opgegeven in de bestaande optieset voor mshr_hcmregulatoryjobcategory.

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | None | Geen. |
| 200000001 | Staf | Functionarissen en managers op executive/senior niveau. |
| 200000002 | Beheerder | Functionarissen en managers op eerste/middenniveau. |
| 200000003 | Beroepsmatig | Professionals. |
| 200000004 | Technicus | Technici. |
| 200000005 | Verkoopmedewerker | Verkoopmedewerkers. |
| 200000006 | Beheer | Administratieve ondersteuningsmedewerkers. |
| 200000007 | Ambachtswerker | Ambachtswerkers. |
| 200000008 | Werker | Werkers. |
| 200000009 | Arbeider | Arbeiders/helpers. |
| 200000010 | Dienstverlener | Dienstverleners. |
