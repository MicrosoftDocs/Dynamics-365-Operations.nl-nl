# <a name="payroll---no-yes-option-set"></a>Salarisadministratie - Optieset Nee Ja

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In dit onderwerp wordt de optieset voor Nee Ja voor Dynamics 365 Human Resources beschreven.

## <a name="option-set-no-yes"></a>Optieset: Nee Ja

Fysieke naam: mshr_noyes

Deze opsomming biedt de optie die is ingesteld voor de standaard Booleaanse eigenschappen in virtuele Human Resources-entiteiten.

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | No | Nee. |
| 200000001 | Ja | Ja. |
