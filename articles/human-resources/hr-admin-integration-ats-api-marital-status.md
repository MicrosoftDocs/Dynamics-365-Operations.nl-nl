# <a name="marital-status"></a>Burgerlijke staat

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In dit onderwerp wordt de optieset voor Burgerlijke staat voor Dynamics 365 Human Resources beschreven.

Fysieke naam: mshr_dirpersonmaritalstatus

Deze opsomming biedt de optieset voor de burgerlijke staat van persoonsrecords.

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | None | Er is geen waarde geselecteerd.
| 200000001 | Ongehuwd | Alleenstaand. |
| 200000002 | Gehuwd | Gehuwd. |
| 200000003 | Gescheiden | Gescheiden. |
| 200000004 | Weduwe/weduwnaar | Weduwe/weduwnaar. |
