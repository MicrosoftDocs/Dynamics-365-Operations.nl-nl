[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="88a28-109">Elk plantype geeft aan of een werknemer zich kan inschrijven voor een bepaald type plan of voor meerdere.</span><span class="sxs-lookup"><span data-stu-id="88a28-109">Each plan type indicates whether an employee can enroll in one plan of its type or multiple.</span></span> <span data-ttu-id="88a28-110">Een werknemer kan zich bijvoorbeeld mogelijk inschrijven voor zowel de verzekeringspolis Basis leven als de verzekeringspolis Aanvullend leven van het plantype Leven.</span><span class="sxs-lookup"><span data-stu-id="88a28-110">For example, an employee would likely be able to enroll in both the Basic life and the Supplemental life policies of plan type Life.</span></span> <span data-ttu-id="88a28-111">Een werknemer mag zich waarschijnlijk inschrijven voor één polis van het type Medisch.</span><span class="sxs-lookup"><span data-stu-id="88a28-111">An employee would likely be allowed to enroll in only one policy of type Medical.</span></span>

<span data-ttu-id="88a28-116">Een plantype kan dekkingsopties definiëren.</span><span class="sxs-lookup"><span data-stu-id="88a28-116">A plan type may define coverage options.</span></span> <span data-ttu-id="88a28-117">De dekkingsopties worden gedefinieerd in het formulier van de optie Behoefteplanning.</span><span class="sxs-lookup"><span data-stu-id="88a28-117">The coverage options are defined in the Coverage option form.</span></span> <span data-ttu-id="88a28-118">Een behoefteplanningsoptie kan het bedrag van de vergoeding aangeven of de contactpersonen die in aanmerking komen voor het plantype.</span><span class="sxs-lookup"><span data-stu-id="88a28-118">A coverage option can specify the amount of the benefit or the contacts who are eligible for the plan type.</span></span> <span data-ttu-id="88a28-119">Als het type contactpersoon bijvoorbeeld Begunstigde is, moet de dekkingsoptie de voorwaarden definiëren van waartoe de begunstigde gerechtigd is als de vergoeding wordt gebruikt.</span><span class="sxs-lookup"><span data-stu-id="88a28-119">For example, if the contact type is Beneficiary, the coverage option should define the terms of what the beneficiary is eligible to receive when the benefit is utilized.</span></span> <span data-ttu-id="88a28-120">Als het contactpersoontype Gezinslid is, moet de dekkingsoptie de relatie tussen het gezinslid en de werknemer definiëren.</span><span class="sxs-lookup"><span data-stu-id="88a28-120">If the contact type is Dependent, the coverage option should define the relationship between the dependent and employee.</span></span>
