# <a name="screening-frequency"></a>Screeningfrequentie

In dit onderwerp wordt de optieset voor Screeningfrequentie voor Dynamics 365 Human Resources beschreven.

Fysieke naam: mshr_hcmfrequencyunit

Voor deze opsomming wordt de optieset met waarden ingesteld voor de screeningfrequentie. 

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 Eenmalig | De screening is slechts eenmaal vereist. |
| 200000001 Dagelijks | De screeningfrequentie wordt berekend in dagen. |
| 200000002 Wekelijks | De screeningfrequentie wordt berekend in weken. |
| 200000003 Maandelijks | De screeningfrequentie wordt berekend in maanden. |
| 200000004 Jaarlijks | De screeningfrequentie wordt berekend in jaren. |
