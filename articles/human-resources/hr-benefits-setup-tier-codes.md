# <a name="set-up-tier-codes"></a>Laagcodes instellen

[!include [banner](includes/preview-feature.md)]

Dynamics 365 Human Resources gebruikt laagcodes met tariefvergoedingen die worden betaald door werknemers en de werk gever. U kunt een onbeperkt aantal laagcodes definiÃ«ren en tariefberekeningen instellen om aan de behoeften van uw bedrijf te voldoen.

1. Selecteer in het werkgebied **Vergoedingenbeheer** onder **Instellen** de optie **Laagcodes**.

2. Selecteer **Nieuw**.

3. Geef de waarden op voor de volgende velden:

   | Veld | Beschrijving |
   | --- | --- |
   | Laagcode | Een unieke code of nummer van een vergoedingslaag die de laagcode identificeert. |
   | Beschrijving | Een beschrijving van de code van een vergoedingslaag. |

4. Selecteer **Opslaan**.
