# <a name="award-type"></a>Toekenningstype


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In dit artikel wordt de optieset voor Toekenningstype voor Dynamics 365 Human Resources beschreven.

## <a name="option-set-award-type"></a>Optieset: Toekenningstype

Fysieke naam: mshr_HrmCompVarAwardEmplType

Deze opsomming biedt de optieset voor het toekenningstype variabele compensatie dat is ingesteld voor standaard Booleaanse eigenschappen in virtuele Human Resources-entiteiten.

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | Algemene toekenning | Algemene toekenning. |
| 200000001 | Totaal bedrag vast loon | Totaal bedrag vast loon. |
