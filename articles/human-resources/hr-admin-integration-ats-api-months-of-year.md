# <a name="months-of-year"></a>Maanden van jaar

Fysieke naam: mshr_monthsofyear

Voor deze opsomming wordt de optieset ingesteld voor de maanden van het jaar.

| 200000005 | Mei | Mei. |
| 200000006 | Juni | Juni. |
| 200000007 | Juli | Juli. |
| 200000008 | Augustus | Augustus. |
| 200000009 | September | September. |
| 200000010 | Oktober | Oktober. |
| 200000011 | November | November. |
| 200000012 | December | December. |
