# <a name="months-of-year"></a>Maanden van jaar

In dit onderwerp wordt de optieset voor Maanden van jaar voor Dynamics 365 Human Resources beschreven.

Fysieke naam: mshr_monthsofyear

Voor deze opsomming wordt de optieset ingesteld voor de maanden van het jaar.

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | None | Er is geen waarde geselecteerd. |
| 200000001 | Januari | Januari. |
| 200000002 | Februari | Februari. |
| 200000003 | Maart | Maart. |
| 200000004 | April | April. |
| 200000005 | Mei | Mei. |
| 200000006 | Juni | Juni. |
| 200000007 | Juli | Juli. |
| 200000008 | Augustus | Augustus. |
| 200000009 | September | September. |
| 200000010 | Oktober | Oktober. |
| 200000011 | November | November. |
| 200000012 | December | December. |
