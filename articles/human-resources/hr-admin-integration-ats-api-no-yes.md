# <a name="no-yes"></a>Nee Ja

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In dit onderwerp wordt de optieset voor Nee Ja voor Dynamics 365 Human Resources beschreven.

## <a name="option-set-no-yes"></a>Optieset: Nee Ja

Fysieke naam: mshr_noyes

Deze opsomming biedt opties die zijn ingesteld voor standaard Booleaanse eigenschappen in virtuele Human Resources-entiteiten.

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | No | Nr. |
| 200000001 | Ja | Ja. |
