# <a name="contact-type"></a>Type contact


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In dit artikel wordt de optieset voor Contacttype voor Dynamics 365 Human Resources beschreven.

Fysieke naam: mshr_logisticselectronicaddressmethodtype

Voor deze opsomming wordt de optieset met waarden ingesteld voor de contacttypen voor de kandidaat. 

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | None | Er is geen type geselecteerd. |
| 200000001 | Telefoonnummer | Telefoonnummer. |
| 200000002 | E-mailadres | E-mailadres. |
| 200000003 | URL | Website-URL. |
| 200000004 | Telex | Telexnummer. |
| 200000005 | Fax | Faxnummer. |
| 200000006 | Facebook | Facebook-account. Geïdentificeerd met Gebruikers-id. |
| 200000007 | Twitter | Twitter-account. Geïdentificeerd met @gebruikersnaam. |
| 200000008 | LinkedIn | LinkedIn-account. Geïdentificeerd met gebruikersnaam. |
