# <a name="custom-fields"></a>Aangepaste velden

U kunt aangepaste velden toevoegen aan pagina's in Dynamics 365 Human Resources. Zie [Maken en werken met aangepaste velden](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields) voor meer informatie over het maken van aangepaste velden.

De meeste Common Data Service-entiteiten ondersteunen aangepaste velden. Een volledige lijst met Common Data Service-entiteiten vindt u in [Common Data Service-entiteiten](https://docs.microsoft.com/dynamics365/human-resources/hr-developer-entities). 

Zorg ervoor dat aangepaste velden die u maakt, worden ingeschakeld, zodat u ze kunt gebruiken in Common Data Service. Zie [Aangepaste velden in gegevensentiteiten weergeven](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities) voor meer informatie.
