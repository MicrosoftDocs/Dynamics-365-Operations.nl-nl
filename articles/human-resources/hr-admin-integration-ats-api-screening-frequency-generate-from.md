# <a name="screening-frequency-generate-from"></a>Screeningfrequentie genereren op basis van


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In dit artikel wordt de optieset voor Screeningfrequentie genereren op basis van voor Dynamics 365 Human Resources beschreven.

Fysieke naam: mshr_hcmfrequencygeneratefrom

Deze opsomming bevat de optieset met waarden om de begindatum van de berekening voor de volgende vereiste screening te bepalen.

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 Niet geselecteerd | Er is geen waarde geselecteerd. |
| 200000001 Datum voltooid | De berekening wordt uitgevoerd vanaf de laatste voltooide screeningdatum. |
| 200000002 Datum vereist | De berekening wordt uitgevoerd vanaf de laatste vereiste screeningdatum. |
