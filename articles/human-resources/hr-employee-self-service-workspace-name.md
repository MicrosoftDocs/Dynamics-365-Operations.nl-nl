# <a name="change-employee-self-service-workspace-name"></a><span data-ttu-id="2d6bb-103">Naam van werkgebied voor Werknemerselfservice wijzigen</span><span class="sxs-lookup"><span data-stu-id="2d6bb-103">Change Employee self service workspace name</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="2d6bb-104">Als u vrijwilligers of andere niet-werknemers hebt, wilt u mogelijk de naam van het werkgebied **Selfservice werknemer** wijzigen.</span><span class="sxs-lookup"><span data-stu-id="2d6bb-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="2d6bb-105">In plaats hiervan kunt u het werkgebied wijzigen in **Selfservice**.</span><span class="sxs-lookup"><span data-stu-id="2d6bb-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="2d6bb-106">Als u de naam van het werkgebied **Selfservice werknemer** wijzigt, verandert ook het menu-item dat intern wordt gebruikt in Dynamics 365 Human Resources.</span><span class="sxs-lookup"><span data-stu-id="2d6bb-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="2d6bb-107">Als u eerder beveiligingsaanpassingen hebt toegepast op de menuopdracht **HcmEmployeeSelfServiceWorkspace**, raden we aan dezelfde wijzigingen toe te passen op **HcmSelfServiceWorkspace** om de overeenkomst te behouden.</span><span class="sxs-lookup"><span data-stu-id="2d6bb-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

3. <span data-ttu-id="2d6bb-110">Selecteer de optie **Selfservice** onder **Weergavenaam**.</span><span class="sxs-lookup"><span data-stu-id="2d6bb-110">Under **Display name**, select **Self service**.</span></span>

   ![Naam van werkgebied voor Werknemerselfservice wijzigen in Selfservice](./media/hr-employee-self-service-workspace-name.png)

4. <span data-ttu-id="2d6bb-112">Selecteer **Opslaan**.</span><span class="sxs-lookup"><span data-stu-id="2d6bb-112">Select **Save**.</span></span>
