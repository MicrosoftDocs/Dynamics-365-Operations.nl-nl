# <a name="run-the-compensation-process"></a>Het compensatieproces uitvoeren

**Probleem**

De klant probeert het compensatieproces in Microsoft Dynamics 365 Human Resources uit te voeren, maar slaagt hier niet in.

**Oorzaak**

De klant heeft de documentatie nodig.

**Resolutie**

Hier is de documentatie voor compensatie:

- Instellen:

    - [Compensatieplannen](https://docs.microsoft.com/dynamics365/unified-operations/talent/compensation-plans)
    - [Plannen voor vaste compensatie maken](https://docs.microsoft.com/dynamics365/unified-operations/talent/create-fixed-compensation-plans)
    - [Plannen voor variabele compensatie maken](https://docs.microsoft.com/dynamics365/unified-operations/talent/create-variable-compensation-plans)

- Proces:

    - [Compensatie verwerken](https://docs.microsoft.com/dynamics365/unified-operations/talent/process-compensation)
