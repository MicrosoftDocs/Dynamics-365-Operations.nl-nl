# <a name="blank-yes-no"></a>Leeg Ja Nee


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In dit artikel wordt de optieset voor Leeg Ja Nee voor Dynamics 365 Human Resources beschreven.

Fysieke naam: mshr_hrmblankyesno

Deze opsomming biedt een optieset voor ja/nee-eigenschappen die ook leeg mogen zijn.

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | Leeg | Er is geen waarde geselecteerd. |
| 200000001 | Ja | Ja. |
| 200000002 | Nee | Nr. |
