# <a name="payroll---employment-type-option-set"></a>Salarisadministratie - Optieset voor type dienstverband

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In dit onderwerp wordt de optieset voor Type dienstverband voor Dynamics 365 Human Resources beschreven.

## <a name="option-set-employment-type"></a>Optieset: type dienstverband

Deze opsomming geeft de optieset type dienstverband voor de werknemer. Deze is beschikbaar in de optieset mshr_hcmemploymenttype.

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | Medewerker | Medewerker |
| 200000001 | Contractant | Contractant |
