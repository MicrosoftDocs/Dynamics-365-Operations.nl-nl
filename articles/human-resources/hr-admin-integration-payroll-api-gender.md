# <a name="payroll---gender-option-set"></a>Salarisadministratie - Optieset Geslacht


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In dit onderwerp wordt de optieset voor Gender voor Dynamics 365 Human Resources beschreven.

## <a name="option-set-gender-option"></a>Optieset: optie Geslacht

Voor deze opsomming wordt de optieset genders ingesteld voor de werknemer. Dit is beschikbaar in de optieset mshr_hcmpersongender.

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | None | Er is geen gender opgegeven. |
| 200000001 | Mannelijk | Man. |
| 200000002 | Vrouwelijk | Vrouw. |
| 200000003 | NonSpecific | Selectie voor een niet-specifiek gender. |
