# <a name="education-credit-basis"></a>Basis van educatief tegoed

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In dit onderwerp wordt de optieset voor de basis van educatief tegoed voor Dynamics 365 Human Resources beschreven.

Fysieke naam: mshr_hcmeducationcreditbasis

Deze opsomming biedt de optieset met waarden voor de basis van het educatieve tegoed voor opleidingsrecord van de kandidaat.

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | Leeg | Er is geen waarde geselecteerd. |
| 200000001 | Semester | Semester. |
| 200000002 | Kwartaal | Kwartaal. |
| 200000003 | Trimester | Trimester. |
| 200000004 | Voorwaarde | Periode. |
| 200000005 | Overige | Overig. |
