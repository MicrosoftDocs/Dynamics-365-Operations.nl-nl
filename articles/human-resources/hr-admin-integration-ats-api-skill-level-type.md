# <a name="skill-level-type"></a>Type vaardigheidsniveau


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In dit onderwerp wordt de optieset voor Type vaardigheidsniveau voor Dynamics 365 Human Resources beschreven.

Fysieke naam: mshr_hrmskillleveltype

Deze opsomming geeft de optieset met waarden die het niveau categoriseren dat aan de vaardigheid van een kandidaat is toegewezen.

| Waarde | Etiket | Beschrijving |
| --- | --- | --- |
| 200000000 | Huidige | Werkelijk vaardigheidsniveau. |
| 200000001 | Doel | Beoogde waarde voor het vaardigheidsniveau. |
