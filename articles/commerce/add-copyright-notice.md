## <a name="add-a-copyright-notice"></a><span data-ttu-id="b6836-109">Een auteursrechtmelding toevoegen</span><span class="sxs-lookup"><span data-stu-id="b6836-109">Add a copyright notice</span></span>

1. <span data-ttu-id="b6836-114">Selecteer **OK**.</span><span class="sxs-lookup"><span data-stu-id="b6836-114">Select **OK**.</span></span>
1. <span data-ttu-id="b6836-115">Selecteer in de navigatiepagina de knop met het weglatingsteken (**...**) naast **Voettekst** en vervolgens **Module toevoegen**.</span><span class="sxs-lookup"><span data-stu-id="b6836-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="b6836-116">Selecteer in het dialoogvenster **Voettekstcategorie** en selecteer vervolgens **OK**.</span><span class="sxs-lookup"><span data-stu-id="b6836-116">In the dialog box, select **Footer category**, and then select **OK**.</span></span>
1. <span data-ttu-id="b6836-117">Selecteer in de navigatiepagina de knop met het weglatingsteken naast **Voettekstcategorie** en vervolgens **Module toevoegen**.</span><span class="sxs-lookup"><span data-stu-id="b6836-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="b6836-118">Selecteer in het dialoogvenster **Blokitem met uitgebreide inhoud**en selecteer vervolgens **OK**.</span><span class="sxs-lookup"><span data-stu-id="b6836-118">In the dialog box, select **Content rich block item**, and then select **OK**.</span></span>
1. <span data-ttu-id="b6836-119">Selecteer in het navigatiedeelvenster de optie **Blokitem met uitgebreide inhoud**.</span><span class="sxs-lookup"><span data-stu-id="b6836-119">In the navigation pane, select **Content rich block item**.</span></span>
1. <span data-ttu-id="b6836-120">Voeg in het deelvenster Eigenschappen rechts in het veld **Alinea** het copyrightbericht toe.</span><span class="sxs-lookup"><span data-stu-id="b6836-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="b6836-121">Voer bijvoorbeeld **(C) Fabrikam 2019** in.</span><span class="sxs-lookup"><span data-stu-id="b6836-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="b6836-122">Selecteer **Opslaan**, **Inchecken** en vervolgens **Publiceren**.</span><span class="sxs-lookup"><span data-stu-id="b6836-122">Select **Save**, select **Check In**, and then select **Publish**.</span></span>
1. <span data-ttu-id="b6836-123">Ga naar **Sjablonen**, selecteer de sjabloon en selecteer **Uitchecken**.</span><span class="sxs-lookup"><span data-stu-id="b6836-123">Go to **Templates**, select the template, and then select **Check Out**.</span></span>
1. <span data-ttu-id="b6836-124">Vouw onder **Paginaoverzicht** **Hoofdtekst** uit en vouw vervolgens **Standaardpagina** uit.</span><span class="sxs-lookup"><span data-stu-id="b6836-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="b6836-125">Selecteer de knop met het weglatingsteken voor het vak **Voettekst** en selecteer **Fragment toevoegen**.</span><span class="sxs-lookup"><span data-stu-id="b6836-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="b6836-126">Selecteer het fragment dat u eerder hebt gemaakt en dan **Selecteren.**</span><span class="sxs-lookup"><span data-stu-id="b6836-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="b6836-127">Check de sjabloon in en publiceer deze.</span><span class="sxs-lookup"><span data-stu-id="b6836-127">Check in the template, and publish it.</span></span>

<span data-ttu-id="b6836-128">De voettekst die de copyrightgegevens bevat, wordt automatisch weergegeven onder aan alle pagina's die de geselecteerde sjabloon gebruiken.</span><span class="sxs-lookup"><span data-stu-id="b6836-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>
