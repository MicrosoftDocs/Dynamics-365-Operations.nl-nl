<span data-ttu-id="69ec8-104">In dit onderwerp wordt beschreven hoe u velden toevoegt aan een Microsoft Excel-werkmap zodat u detailhandelstransacties in Microsoft Dynamics 365 Commerce kunt bewerken.</span><span class="sxs-lookup"><span data-stu-id="69ec8-104">This topic describes how to add fields to a Microsoft Excel workbook so that you can edit retail transactions in Microsoft Dynamics 365 Commerce.</span></span>

## <a name="overview"></a><span data-ttu-id="69ec8-105">Overzicht</span><span class="sxs-lookup"><span data-stu-id="69ec8-105">Overview</span></span>

<span data-ttu-id="69ec8-106">Wanneer u een Excel-bestand genereert, zodat u detailhandelstransacties kunt bewerken, wordt het bestand gevuld met een aantal standaardvelden.</span><span class="sxs-lookup"><span data-stu-id="69ec8-106">When you generate an Excel file so that you can edit retail transactions, the file is filled with some default fields.</span></span> <span data-ttu-id="69ec8-107">Als een veld dat moet worden bijgewerkt, standaard niet zichtbaar is in het gegenereerde Excel-bestand, kunt u het toevoegen.</span><span class="sxs-lookup"><span data-stu-id="69ec8-107">If a field that must be updated isn't visible by default in the generated Excel file, you can add it.</span></span>

## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a><span data-ttu-id="69ec8-108">Velden toevoegen aan een werkblad in een Excel-werkmap</span><span class="sxs-lookup"><span data-stu-id="69ec8-108">Add fields to a worksheet in an Excel workbook</span></span>

<span data-ttu-id="69ec8-109">Voer de volgende stappen uit om velden toe te voegen aan een Excel-werkmap zodat u detailhandelstransacties kunt bewerken.</span><span class="sxs-lookup"><span data-stu-id="69ec8-109">To add fields to an Excel workbook so that you can edit retail transactions, follow these steps.</span></span>

1. <span data-ttu-id="69ec8-110">Download en open het Excel-bestand op de pagina **Overzichten**, de pagina **Detailhandelstransacties** of de tegel **Fouten in transactievalidatie** in het werkgebied **Financiën van winkel**.</span><span class="sxs-lookup"><span data-stu-id="69ec8-110">Download and open the Excel file from the **Statements** page, the **Retail transactions** page, or the **Transaction validation failures** tile in the **Store financials** workspace.</span></span>
1. <span data-ttu-id="69ec8-111">Selecteer **Ontwerpen**.</span><span class="sxs-lookup"><span data-stu-id="69ec8-111">Select **Design**.</span></span>
1. <span data-ttu-id="69ec8-112">Selecteer het potloodsymbool voor de gewenste tabel en zoek en selecteer vervolgens in de lijst met beschikbare velden het veld dat u wilt toevoegen.</span><span class="sxs-lookup"><span data-stu-id="69ec8-112">Select the pencil symbol for the desired table, and then, in the list of available fields, find and select the field that you want to add.</span></span>
1. <span data-ttu-id="69ec8-113">Selecteer **Toevoegen** en vervolgens **Bijwerken**.</span><span class="sxs-lookup"><span data-stu-id="69ec8-113">Select **Add**, and then select **Update**.</span></span> <span data-ttu-id="69ec8-114">U kunt de volgorde van de velden wijzigen.</span><span class="sxs-lookup"><span data-stu-id="69ec8-114">You can reorder fields.</span></span>
1. <span data-ttu-id="69ec8-115">Nadat de update is voltooid, selecteert u **Vernieuwen** om de gegevens voor de nieuwe kolom op te halen.</span><span class="sxs-lookup"><span data-stu-id="69ec8-115">After the update is completed, select **Refresh** to fetch the data for the new column.</span></span>

<span data-ttu-id="69ec8-116">Het nieuwe veld en de gegevens daarvoor moeten nu beschikbaar zijn voor bewerking in Excel.</span><span class="sxs-lookup"><span data-stu-id="69ec8-116">The new field and data for it should now be available for editing in Excel.</span></span>
