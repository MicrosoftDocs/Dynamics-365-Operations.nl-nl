    <span data-ttu-id="c0ef7-121">Meldingen worden alleen weergegeven voor bewerkingen die zijn toegevoegd op het sneltabblad **Meldingen** en u kunt alleen bewerkingen toevoegen als het selectievakje **Meldingen inschakelen** voor deze bewerkingen is geselecteerd op de pagina **POS-bewerkingen**.</span><span class="sxs-lookup"><span data-stu-id="c0ef7-121">Notifications are shown only for operations that are added on the **Notifications** FastTab, and you can add operations there only if the **Enable notifications** check box for those operations has been selected on the **POS operations** page.</span></span> <span data-ttu-id="c0ef7-122">Bovendien worden meldingen voor een bewerking alleen aan werknemers weergegeven als de bewerking wordt toegevoegd aan de POS-machtigingen voor die werknemers.</span><span class="sxs-lookup"><span data-stu-id="c0ef7-122">Additionally, notifications for an operation are shown to workers only if the operation is added to the POS permissions for those workers.</span></span>
