## <a name="crop-an-image"></a><span data-ttu-id="f51ea-107">Een afbeelding bijsnijden</span><span class="sxs-lookup"><span data-stu-id="f51ea-107">Crop an image</span></span>

<span data-ttu-id="f51ea-108">Volg deze stappen om een afbeelding bij te snijden in Site Builder.</span><span class="sxs-lookup"><span data-stu-id="f51ea-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="f51ea-109">Selecteer **Mediabibliotheek** in het linkernavigatievenster van Commerce Site Builder.</span><span class="sxs-lookup"><span data-stu-id="f51ea-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="f51ea-110">Selecteer in het hoofdvenster de afbeelding die u wilt wijzigen.</span><span class="sxs-lookup"><span data-stu-id="f51ea-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="f51ea-111">Selecteer **Bewerken** op de opdrachtbalk.</span><span class="sxs-lookup"><span data-stu-id="f51ea-111">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="f51ea-112">Selecteer de afbeelding om de **Bewerkingsmodus** te activeren.</span><span class="sxs-lookup"><span data-stu-id="f51ea-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="f51ea-113">Selecteer onder **Bewerkingsmodus** de optie **Weergave bewerken per module**.</span><span class="sxs-lookup"><span data-stu-id="f51ea-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="f51ea-114">Selecteer in de vervolgkeuzelijst **Module** het moduletype.</span><span class="sxs-lookup"><span data-stu-id="f51ea-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="f51ea-115">Selecteer in de vervolgkeuzelijst **Weergavetype** het weergavetype.</span><span class="sxs-lookup"><span data-stu-id="f51ea-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="f51ea-116">Selecteer in het vervolgkeuzemenu **Plaatsing** een plaatsingsoptie voor de afbeelding.</span><span class="sxs-lookup"><span data-stu-id="f51ea-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="f51ea-117">Selecteer in de vervolgkeuzelijst **Viewport** de afmetingen van de viewport.</span><span class="sxs-lookup"><span data-stu-id="f51ea-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="f51ea-118">Op de afbeelding verschijnt een overlay met het gebied dat het snijgebied aangeeft.</span><span class="sxs-lookup"><span data-stu-id="f51ea-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="f51ea-119">Verplaats het snijgebied naar wens en wijzig de afmetingen.</span><span class="sxs-lookup"><span data-stu-id="f51ea-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="f51ea-120">De hoogte-breedteverhouding wordt automatisch behouden.</span><span class="sxs-lookup"><span data-stu-id="f51ea-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="f51ea-121">Wanneer u klaar bent, selecteert u op de opdrachtbalk de optie **Opslaan** en vervolgens **Bewerken voltooien**.</span><span class="sxs-lookup"><span data-stu-id="f51ea-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="f51ea-122">Nadat de afbeelding is bijgesneden, worden de wijzigingen in de afbeelding vrijwel onmiddellijk van kracht.</span><span class="sxs-lookup"><span data-stu-id="f51ea-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
