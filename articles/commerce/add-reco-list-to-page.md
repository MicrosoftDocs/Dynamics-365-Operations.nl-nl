# <a name="add-recommendation-lists-to-a-site"></a>Aanbevelingslijsten toevoegen aan een site


[!include [banner](includes/banner.md)]

In dit onderwerp wordt uitgelegd hoe u een lijst met productaanbevelingen aan een e-commerce-pagina toevoegt.

1. Selecteer **Module toevoegen** op een bestaande pagina e-Commerce Site Builder.
2. Zoeken de module **productverzameling**.
3. Selecteer in de modulewizard de lijst die u aan de pagina wilt toevoegen.
4. Sla de wijzigingen op.

Zie [Overzicht van productcollectiemodules](product-collection-module-overview.md) voor meer informatie over de lijsttypen die u aan een pagina kunt toevoegen.
