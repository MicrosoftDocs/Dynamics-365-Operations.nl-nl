# <a name="add-product-recommendation-lists-to-e-commerce-pages"></a>Lijsten met productaanbevelingen toevoegen aan e-commerce-pagina's

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

In dit onderwerp wordt uitgelegd hoe u een lijst met productaanbevelingen aan een e-commerce-pagina toevoegt.

1. Selecteer **Module toevoegen** op een bestaande e-commerce-pagina voor ontwerpen.
2. Zoeken de module **productverzameling**.
3. Selecteer in de modulewizard de lijst die u aan de pagina wilt toevoegen.
4. Sla de wijzigingen op.

Zie [Overzicht met productaanbevelingen](product-recommendations.md) voor meer informatie over de lijsttypen die u aan een pagina kunt toevoegen.
