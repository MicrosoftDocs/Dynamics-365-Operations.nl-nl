# <a name="customize-image-focal-points"></a><span data-ttu-id="8a672-103">De focuspunten van de afbeelding aanpassen</span><span class="sxs-lookup"><span data-stu-id="8a672-103">Customize image focal points</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="8a672-104">In dit onderwerp wordt beschreven hoe u de focuspunten van afbeeldingen aanpast in Microsoft Dynamics 365 Commerce Site Builder.</span><span class="sxs-lookup"><span data-stu-id="8a672-104">This topic describes how to customize image focal points in Microsoft Dynamics 365 Commerce site builder.</span></span>

## <a name="overview"></a><span data-ttu-id="8a672-105">Overzicht</span><span class="sxs-lookup"><span data-stu-id="8a672-105">Overview</span></span>

<span data-ttu-id="8a672-106">Wanneer een afbeelding wordt geüpload naar mediabibliotheek in Commerce Site Builder, probeert het systeem het focuspunt van de afbeelding te bepalen.</span><span class="sxs-lookup"><span data-stu-id="8a672-106">When an image is uploaded to the Commerce site builder Media Library, the system attempts to determine the focal point of the image.</span></span> <span data-ttu-id="8a672-107">Als de afbeelding bijvoorbeeld een persoon bevat, wordt het focuspunt door het systeem standaard ingesteld op het gezicht van de persoon.</span><span class="sxs-lookup"><span data-stu-id="8a672-107">For example, if the image has a person on it, the system will set the focal point to the face of the person by default.</span></span> <span data-ttu-id="8a672-108">In de meeste gevallen is het automatisch instellen van een focuspunt geschikt voor alle viewports, maar soms wilt u het focuspunt aanpassen om ervoor te zorgen dat een bepaald deel van de afbeelding altijd zichtbaar is.</span><span class="sxs-lookup"><span data-stu-id="8a672-108">In most cases the automatically set focal point works well for all viewports, but sometimes you may want to adjust the focal point to ensure that a specific part of the image is always visible.</span></span>

### <a name="define-a-custom-focal-point-for-an-image"></a><span data-ttu-id="8a672-109">Een aangepast focuspunt voor een afbeelding definiëren</span><span class="sxs-lookup"><span data-stu-id="8a672-109">Define a custom focal point for an image</span></span>

<span data-ttu-id="8a672-110">Voer de volgende stappen uit om een aangepast focuspunt te definiëren voor een afbeelding.</span><span class="sxs-lookup"><span data-stu-id="8a672-110">To define a custom focal point for an image, follow these steps.</span></span>

1. <span data-ttu-id="8a672-111">Selecteer **Mediabibliotheek** in het linkernavigatievenster van Commerce Site Builder.</span><span class="sxs-lookup"><span data-stu-id="8a672-111">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="8a672-112">Selecteer in het hoofdvenster de afbeelding die u wilt wijzigen.</span><span class="sxs-lookup"><span data-stu-id="8a672-112">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="8a672-113">Selecteer **Bewerken** op de opdrachtbalk.</span><span class="sxs-lookup"><span data-stu-id="8a672-113">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="8a672-114">Selecteer de afbeelding om de **Bewerkingsmodus** te activeren.</span><span class="sxs-lookup"><span data-stu-id="8a672-114">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="8a672-115">Selecteer onder **Bewerkingsmodus** de optie **Focuspunt wijzigen**.</span><span class="sxs-lookup"><span data-stu-id="8a672-115">Under **Edit Mode**, select **Change Focal Point**.</span></span> <span data-ttu-id="8a672-116">Er wordt een cirkelvormig focuspunt boven de afbeelding weer gegeven.</span><span class="sxs-lookup"><span data-stu-id="8a672-116">A circular focal point control appears over the image.</span></span>
1. <span data-ttu-id="8a672-117">Selecteer de focuspuntbesturing om deze over het gewenste focuspunt te bewegen.</span><span class="sxs-lookup"><span data-stu-id="8a672-117">Select the focal point control to move it over the desired focal point.</span></span>
1. <span data-ttu-id="8a672-118">Wanneer u klaar bent, selecteert u op de opdrachtbalk de optie **Opslaan** en vervolgens **Bewerken voltooien**.</span><span class="sxs-lookup"><span data-stu-id="8a672-118">When you're done, on the command bar select **Save**, and then select **Finish editing**.</span></span>
