# <a name="configure-the-functionality-profile-for-a-sales-representative"></a><span data-ttu-id="41b78-103"> Het functionaliteitsprofiel voor een verkoopvertegenwoordiger configureren</span><span class="sxs-lookup"><span data-stu-id="41b78-103">Configure the functionality profile for a sales representative</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="41b78-104">Deze procedure geeft aan hoe u de instellingen van het functionaliteitsprofiel van een winkel configureert dat van toepassing is op verkopers die het demobedrijf USRT gebruiken.</span><span class="sxs-lookup"><span data-stu-id="41b78-104">This procedure demonstrates how to configure a store's functionality profile settings that apply to sales representatives using the USRT demo data company.</span></span>

1. <span data-ttu-id="41b78-105">Ga naar Retail en Commerce > Kanaalinstellingen > POS-instellingen > POS-profielen > Functionaliteitsprofielen.</span><span class="sxs-lookup"><span data-stu-id="41b78-105">Go to Retail and Commerce > Channel setup > POS setup > POS profiles > Functionality profiles.</span></span>
2. <span data-ttu-id="41b78-106">Klik op Bewerken.</span><span class="sxs-lookup"><span data-stu-id="41b78-106">Click Edit.</span></span>
3. <span data-ttu-id="41b78-107">Vouw de sectie Functies uit.</span><span class="sxs-lookup"><span data-stu-id="41b78-107">Expand the Functions section.</span></span>
    * <span data-ttu-id="41b78-108">U kunt de instellingen van het functionaliteitprofiel gebruiken om POS te configureren op het automatisch toevoegen van de standaardverkoopgroep van de kassier, op vragen om verkoopgroepen en verkoopgroepen vereisen.</span><span class="sxs-lookup"><span data-stu-id="41b78-108">You can use the functionality profile settings to configure POS to automatically add the cashier's default sales group, to prompt for sales groups, and to require sales groups.</span></span>  
4. <span data-ttu-id="41b78-109">Selecteer een optie in het veld Vragen om verkoopvertegenwoordiger.</span><span class="sxs-lookup"><span data-stu-id="41b78-109">In the Prompt for sales representative field, select an option.</span></span>
5. <span data-ttu-id="41b78-110">Selecteer Ja in het veld Verkoopvertegenwoordiger vereisen.</span><span class="sxs-lookup"><span data-stu-id="41b78-110">Select Yes in the Require sales representative field.</span></span>
