<span data-ttu-id="0dddc-109">Volg deze stappen om een functionaliteitsprofiel te maken.</span><span class="sxs-lookup"><span data-stu-id="0dddc-109">To create a functionality profile, follow these steps.</span></span>

1. <span data-ttu-id="0dddc-118">Pas in de sectie **Bedrag** zo nodig instellingen aan.</span><span class="sxs-lookup"><span data-stu-id="0dddc-118">In the **Amount** section, modify settings as, needed.</span></span>
1. <span data-ttu-id="0dddc-119">Pas in de sectie **Infocodes** zo nodig instellingen aan.</span><span class="sxs-lookup"><span data-stu-id="0dddc-119">In the **Info Codes** section, modify settings, as needed.</span></span>
