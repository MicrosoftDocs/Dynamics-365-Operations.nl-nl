## <a name="create-a-functionality-profile"></a><span data-ttu-id="4384b-107">Een functionaliteitsprofiel maken</span><span class="sxs-lookup"><span data-stu-id="4384b-107">Create a functionality profile</span></span>

<span data-ttu-id="4384b-108">Volg deze stappen om een functionaliteitsprofiel te maken.</span><span class="sxs-lookup"><span data-stu-id="4384b-108">To create a functionality profile, follow these steps.</span></span>

1. <span data-ttu-id="4384b-109">Ga in het navigatievenster naar **Modules \> Kanaalinstelling \> POS-profielen \> Functionaliteitsprofielen**.</span><span class="sxs-lookup"><span data-stu-id="4384b-109">In the navigation pane, go to **Modules \> Channel setup \> POS profiles \> Functionality profiles**.</span></span>
1. <span data-ttu-id="4384b-110">Selecteer **Nieuw** in het actievenster.</span><span class="sxs-lookup"><span data-stu-id="4384b-110">On the action pane, select **New**.</span></span>
1. <span data-ttu-id="4384b-111">Voer in het veld **Profiel** een id in voor het profiel ('FN006' in de voorbeeldafbeelding hieronder).</span><span class="sxs-lookup"><span data-stu-id="4384b-111">In the **Profile** field, enter an ID for the profile ("FN006" in the example image below).</span></span>
1. <span data-ttu-id="4384b-112">Voer in het veld **Beschrijving** een waarde in ('Profiel van Adventure Works' in de voorbeeldafbeelding hieronder).</span><span class="sxs-lookup"><span data-stu-id="4384b-112">In the **Description** field, enter a value ("Adventure Works Profile" in the example image below).</span></span>
1. <span data-ttu-id="4384b-113">Selecteer in de sectie **Algemeen** een land voor de landinstelling **ISO**.</span><span class="sxs-lookup"><span data-stu-id="4384b-113">In the **General** section, select a country for the **ISO** locale.</span></span>
1. <span data-ttu-id="4384b-114">Pas in de sectie **Algemeen** zo nodig de overige instellingen aan.</span><span class="sxs-lookup"><span data-stu-id="4384b-114">In the **General** section, modify other settings, as needed.</span></span>
1. <span data-ttu-id="4384b-115">Selecteer in de sectie **Algemeen** een **id ontvangstbewijsprofiel** voor ontvangstbewijzen per e-mail.</span><span class="sxs-lookup"><span data-stu-id="4384b-115">In the **General** section, select a **Receipt profile ID** for email receipts.</span></span>
1. <span data-ttu-id="4384b-116">Pas in de sectie **Functies** zo nodig instellingen aan.</span><span class="sxs-lookup"><span data-stu-id="4384b-116">In the **Functions** section, modify settings, as needed.</span></span>
1. <span data-ttu-id="4384b-117">Pas in de sectie **Bedrag** zo nodig instellingen aan.</span><span class="sxs-lookup"><span data-stu-id="4384b-117">In the **Amount** section, modify settings as, needed.</span></span>
1. <span data-ttu-id="4384b-118">Pas in de sectie **Infocodes** zo nodig instellingen aan.</span><span class="sxs-lookup"><span data-stu-id="4384b-118">In the **Info Codes** section, modify settings, as needed.</span></span>
1. <span data-ttu-id="4384b-119">Pas in de sectie **Ontvangstbewijsnummering** zo nodig de overige instellingen aan.</span><span class="sxs-lookup"><span data-stu-id="4384b-119">In the **Receipt numbering** section, modify settings, as needed.</span></span>
