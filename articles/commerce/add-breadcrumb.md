<span data-ttu-id="ee239-106">Breadcrumb-modules worden gebruikt om secundaire navigatie uit te voeren op sitepagina's.</span><span class="sxs-lookup"><span data-stu-id="ee239-106">Breadcrumb modules are used to provide secondary navigation on site pages.</span></span> <span data-ttu-id="ee239-107">Ze worden meestal boven aan een pagina weer gegeven, onder de koptekst.</span><span class="sxs-lookup"><span data-stu-id="ee239-107">They are typically shown at the top of a page, below the header.</span></span> <span data-ttu-id="ee239-108">Hoewel u breadcrumb-modules aan elke pagina kunt toevoegen, worden ze meestal op de pagina's met productgegevens (PDP's) gebruikt om de hiërarchie van productcategorieën weer te geven zodat u snel door een locatie kunt navigeren.</span><span class="sxs-lookup"><span data-stu-id="ee239-108">Although breadcrumb modules can be added to any page, they are most often used on product details pages (PDPs), to show the product category hierarchy and provide a quick way to move around a site.</span></span> <span data-ttu-id="ee239-109">Een breadcrumb-module kan ook worden gebruikt om de koppeling Terug naar resultaten weer te geven wanneer gebruikers een PDP openen vanuit een zoek- of lijstpagina.</span><span class="sxs-lookup"><span data-stu-id="ee239-109">A breadcrumb module can also be used to show a "Back to results" link when users open a PDP from a search or list page.</span></span> <span data-ttu-id="ee239-110">Op deze manier kunnen gebruikers snel terugkeren naar de gefilterde lijstpagina om door te gaan met winkelen.</span><span class="sxs-lookup"><span data-stu-id="ee239-110">In this way, users can quickly return to their filtered list page to continue shopping.</span></span>

## <a name="breadcrumb-module-settings"></a><span data-ttu-id="ee239-117">Instellingen van breadcrumb-module</span><span class="sxs-lookup"><span data-stu-id="ee239-117">Breadcrumb module settings</span></span>

<span data-ttu-id="ee239-118">De breadcrumb-module is gebaseerd op de instelling **Breadcrumb-weergavetype voor PDP** die wordt gedefinieerd bij **Site-instellingen \> Uitbreidingen** in Site builder.</span><span class="sxs-lookup"><span data-stu-id="ee239-118">The breadcrumb module relies on the **Breadcrumb display type on PDP** setting, which is defined at **Site Settings \> Extensions** in site builder.</span></span> <span data-ttu-id="ee239-119">Voor deze instelling bestaan drie mogelijke waarden:</span><span class="sxs-lookup"><span data-stu-id="ee239-119">This setting has three possible values:</span></span>

- <span data-ttu-id="ee239-120">**Categoriehiërarchie weergeven**: wanneer deze waarde wordt geselecteerd, wordt in de breadcrumb-module de volledige categoriehiërarchie weergegeven van het product dat op de PDP wordt weergegeven.</span><span class="sxs-lookup"><span data-stu-id="ee239-120">**Show category hierarchy** – When this value is selected, the breadcrumb module will show the full category hierarchy of the product that is viewed on the PDP.</span></span>
- <span data-ttu-id="ee239-121">**Terug naar resultaten weergeven**: wanneer deze waarde wordt geselecteerd, wordt in de breadcrumb-module de koppeling "Terug naar resultaten" weergegeven op een PDP als de gebruiker de PDP opent vanuit een module die de koppeling "Terug naar resultaten" toestaat.</span><span class="sxs-lookup"><span data-stu-id="ee239-121">**Show back to results** – When this value is selected, the breadcrumb module will show a "Back to results" link on a PDP if the user opened the PDP from a module that allows for a "Back to results" link.</span></span> <span data-ttu-id="ee239-122">Deze functie is beschikbaar wanneer gebruikers navigeren vanuit de pagina's categorie, zoeken, lijst en aanbevelingslijsten.</span><span class="sxs-lookup"><span data-stu-id="ee239-122">This functionality is available when users navigate from category, search, list, and recommendation lists pages.</span></span> <span data-ttu-id="ee239-123">Ter ondersteuning van deze functionaliteit hebben modules voor productverzameling en zoekresultaten een eigenschap met de naam **Terug naar resultaten van PDP toestaan**.</span><span class="sxs-lookup"><span data-stu-id="ee239-123">To support this functionality, product collection and search results modules have a property that is named **Allow back to results on PDP**.</span></span> <span data-ttu-id="ee239-124">Deze eigenschap geeft u de flexibiliteit om te bepalen welke modules de koppelingsfunctionaliteit "Terug naar resultaten" op de PDP moeten ondersteunen.</span><span class="sxs-lookup"><span data-stu-id="ee239-124">This property gives you the flexibility to define which modules should support the "Back to results" link functionality on the PDP.</span></span> <span data-ttu-id="ee239-125">Wanneer bijvoorbeeld **Terug naar resultaten weergeven** wordt geselecteerd voor de instelling **Breadcrumb-weergavetype voor PDP** van de breadcrumb-module, en **Terug naar resultaten van PDP toestaan** is geselecteerd voor de zoekmodule van de zoekpagina, wordt de koppeling Terug naar resultaten weergegeven wanneer gebruikers navigeren vanaf de zoekpagina naar een PDP.</span><span class="sxs-lookup"><span data-stu-id="ee239-125">For example, when **Show back to results** is selected for the **Breadcrumb display type on PDP** setting of the breadcrumb module, and **Allow back to results on PDP** is selected for the search page search results module, a "Back to results" link will be shown when users navigate from the search page to a PDP.</span></span>
- <span data-ttu-id="ee239-126">**Categoriehiërarchie weergeven en terug naar resultaten**: deze waarde is een combinatie van de voorgaande twee.</span><span class="sxs-lookup"><span data-stu-id="ee239-126">**Show category hierarchy and back to results** – This value is a combination of the previous two.</span></span> <span data-ttu-id="ee239-127">Wanneer deze waarde wordt geselecteerd, worden in de breadcrumb-module zowel de hiërarchie van de volledige categorie als de koppeling Terug naar resultaten weergegeven (als deze is geconfigureerd) op een PDP.</span><span class="sxs-lookup"><span data-stu-id="ee239-127">When this value is selected, the breadcrumb module will show both the full category hierarchy and a "Back to results" link (if it's configured) on a PDP.</span></span>

> <span data-ttu-id="ee239-128">Deze instellingen zijn beschikbaar in Dynamics 365 Commerce versie 10.0.12.</span><span class="sxs-lookup"><span data-stu-id="ee239-128">These settings are available in the Dynamics 365 Commerce 10.0.12 release.</span></span> <span data-ttu-id="ee239-129">Als u een oudere versie van Dynamics 365 Commerce bijwerkt, moet u het bestand appsettings.json handmatig bijwerken.</span><span class="sxs-lookup"><span data-stu-id="ee239-129">If you are updating from an older version of Dynamics 365 Commerce, you must manually update the appsettings.json file.</span></span> <span data-ttu-id="ee239-130">Zie [Updates voor SDK's en modulebibliotheken](e-commerce-extensibility/sdk-updates.md#update-the-appsettingsjson-file) voor instructies voor het bijwerken van het appsettings.json.</span><span class="sxs-lookup"><span data-stu-id="ee239-130">For instructions on updating the appsettings.json file, see [SDK and module library updates](e-commerce-extensibility/sdk-updates.md#update-the-appsettingsjson-file).</span></span>
