# <a name="breadcrumb-module"></a><span data-ttu-id="aa002-103">Breadcrumb-module</span><span class="sxs-lookup"><span data-stu-id="aa002-103">Breadcrumb module</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="aa002-104">In dit onderwerp wordt beschreven wat breadcrumb-modules zijn en hoe u ze toevoegt aan sitepagina's in Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="aa002-104">This topic covers breadcrumb modules and describes how to add them to site pages in Microsoft Dynamics 365 Commerce.</span></span>

<span data-ttu-id="aa002-105">Breadcrumb-modules worden gebruikt om secundaire navigatie uit te voeren op sitepagina's.</span><span class="sxs-lookup"><span data-stu-id="aa002-105">Breadcrumb modules are used to provide secondary navigation on site pages.</span></span> <span data-ttu-id="aa002-106">Ze worden meestal boven aan een pagina weer gegeven, onder de koptekst.</span><span class="sxs-lookup"><span data-stu-id="aa002-106">They are typically shown at the top of a page, below the header.</span></span> <span data-ttu-id="aa002-107">Hoewel u breadcrumb-modules aan elke pagina kunt toevoegen, worden ze meestal op de pagina's met productgegevens (PDP's) gebruikt om de hiërarchie van productcategorieën weer te geven zodat u snel door een locatie kunt navigeren.</span><span class="sxs-lookup"><span data-stu-id="aa002-107">Although breadcrumb modules can be added to any page, they are most often used on product details pages (PDPs), to show the product category hierarchy and provide a quick way to move around a site.</span></span> <span data-ttu-id="aa002-108">Een breadcrumb-module kan ook worden gebruikt om de koppeling Terug naar resultaten weer te geven wanneer gebruikers een PDP openen vanuit een zoek- of lijstpagina.</span><span class="sxs-lookup"><span data-stu-id="aa002-108">A breadcrumb module can also be used to show a "Back to results" link when users open a PDP from a search or list page.</span></span> <span data-ttu-id="aa002-109">Op deze manier kunnen gebruikers snel terugkeren naar de gefilterde lijstpagina om door te gaan met winkelen.</span><span class="sxs-lookup"><span data-stu-id="aa002-109">In this way, users can quickly return to their filtered list page to continue shopping.</span></span>

## <a name="breadcrumb-module-settings"></a><span data-ttu-id="aa002-116">Instellingen van breadcrumb-module</span><span class="sxs-lookup"><span data-stu-id="aa002-116">Breadcrumb module settings</span></span>

<span data-ttu-id="aa002-117">De breadcrumb-module is gebaseerd op de instelling **Breadcrumb-weergavetype voor PDP** die wordt gedefinieerd bij **Site-instellingen \> Uitbreidingen** in Site builder.</span><span class="sxs-lookup"><span data-stu-id="aa002-117">The breadcrumb module relies on the **Breadcrumb display type on PDP** setting, which is defined at **Site Settings \> Extensions** in site builder.</span></span> <span data-ttu-id="aa002-118">Voor deze instelling bestaan drie mogelijke waarden:</span><span class="sxs-lookup"><span data-stu-id="aa002-118">This setting has three possible values:</span></span>

- <span data-ttu-id="aa002-119">**Categoriehiërarchie weergeven**: wanneer deze waarde wordt geselecteerd, wordt in de breadcrumb-module de volledige categoriehiërarchie weergegeven van het product dat op de PDP wordt weergegeven.</span><span class="sxs-lookup"><span data-stu-id="aa002-119">**Show category hierarchy** – When this value is selected, the breadcrumb module will show the full category hierarchy of the product that is viewed on the PDP.</span></span>
- <span data-ttu-id="aa002-120">**Terug naar resultaten weergeven**: wanneer deze waarde wordt geselecteerd, wordt in de breadcrumb-module de koppeling "Terug naar resultaten" weergegeven op een PDP als de gebruiker de PDP opent vanuit een module die de koppeling "Terug naar resultaten" toestaat.</span><span class="sxs-lookup"><span data-stu-id="aa002-120">**Show back to results** – When this value is selected, the breadcrumb module will show a "Back to results" link on a PDP if the user opened the PDP from a module that allows for a "Back to results" link.</span></span> <span data-ttu-id="aa002-121">Deze functie is beschikbaar wanneer gebruikers navigeren vanuit de pagina's categorie, zoeken, lijst en aanbevelingslijsten.</span><span class="sxs-lookup"><span data-stu-id="aa002-121">This functionality is available when users navigate from category, search, list, and recommendation lists pages.</span></span> <span data-ttu-id="aa002-122">Ter ondersteuning van deze functionaliteit hebben modules voor productverzameling en zoekresultaten een eigenschap met de naam **Terug naar resultaten van PDP toestaan**.</span><span class="sxs-lookup"><span data-stu-id="aa002-122">To support this functionality, product collection and search results modules have a property that is named **Allow back to results on PDP**.</span></span> <span data-ttu-id="aa002-123">Deze eigenschap geeft u de flexibiliteit om te bepalen welke modules de koppelingsfunctionaliteit "Terug naar resultaten" op de PDP moeten ondersteunen.</span><span class="sxs-lookup"><span data-stu-id="aa002-123">This property gives you the flexibility to define which modules should support the "Back to results" link functionality on the PDP.</span></span> <span data-ttu-id="aa002-124">Wanneer bijvoorbeeld **Terug naar resultaten weergeven** wordt geselecteerd voor de instelling **Breadcrumb-weergavetype voor PDP** van de breadcrumb-module, en **Terug naar resultaten van PDP toestaan** is geselecteerd voor de zoekmodule van de zoekpagina, wordt de koppeling Terug naar resultaten weergegeven wanneer gebruikers navigeren vanaf de zoekpagina naar een PDP.</span><span class="sxs-lookup"><span data-stu-id="aa002-124">For example, when **Show back to results** is selected for the **Breadcrumb display type on PDP** setting of the breadcrumb module, and **Allow back to results on PDP** is selected for the search page search results module, a "Back to results" link will be shown when users navigate from the search page to a PDP.</span></span>
- <span data-ttu-id="aa002-125">**Categoriehiërarchie weergeven en terug naar resultaten**: deze waarde is een combinatie van de voorgaande twee.</span><span class="sxs-lookup"><span data-stu-id="aa002-125">**Show category hierarchy and back to results** – This value is a combination of the previous two.</span></span> <span data-ttu-id="aa002-126">Wanneer deze waarde wordt geselecteerd, worden in de breadcrumb-module zowel de hiërarchie van de volledige categorie als de koppeling Terug naar resultaten weergegeven (als deze is geconfigureerd) op een PDP.</span><span class="sxs-lookup"><span data-stu-id="aa002-126">When this value is selected, the breadcrumb module will show both the full category hierarchy and a "Back to results" link (if it's configured) on a PDP.</span></span>

> [!IMPORTANT]
> <span data-ttu-id="aa002-127">Deze instellingen zijn beschikbaar in Dynamics 365 Commerce versie 10.0.12.</span><span class="sxs-lookup"><span data-stu-id="aa002-127">These settings are available in the Dynamics 365 Commerce 10.0.12 release.</span></span> <span data-ttu-id="aa002-128">Als u een oudere versie van Dynamics 365 Commerce bijwerkt, moet u het bestand appsettings.json handmatig bijwerken.</span><span class="sxs-lookup"><span data-stu-id="aa002-128">If you are updating from an older version of Dynamics 365 Commerce, you must manually update the appsettings.json file.</span></span> <span data-ttu-id="aa002-129">Zie [Updates voor SDK's en modulebibliotheken](e-commerce-extensibility/sdk-updates.md#update-the-appsettingsjson-file) voor instructies voor het bijwerken van het appsettings.json.</span><span class="sxs-lookup"><span data-stu-id="aa002-129">For instructions on updating the appsettings.json file, see [SDK and module library updates](e-commerce-extensibility/sdk-updates.md#update-the-appsettingsjson-file).</span></span>

<span data-ttu-id="aa002-143">Voer de volgende stappen uit om een breadcrumb-module aan een PDP toe te voegen en de vereiste eigenschappen in te stellen.</span><span class="sxs-lookup"><span data-stu-id="aa002-143">To add a breadcrumb module to a PDP and set the required properties, follow these steps.</span></span>

1. <span data-ttu-id="aa002-144">Ga naar **Site-instellingen \> Uitbreidingen** en selecteer vervolgens voor **Breadcrumb-weergavetype voor PDP** de optie **Categoriehiërarchie weergeven**.</span><span class="sxs-lookup"><span data-stu-id="aa002-144">Go to **Site Settings \> Extensions**, and then, for the **Breadcrumb display type on PDP** setting, select **Show category hierarchy**.</span></span>

1. <span data-ttu-id="aa002-146">Selecteer in het vak **Container** dat de koopvakmodule bevat, het weglatingsteken (**...**) en selecteer vervolgens **Module toevoegen**.</span><span class="sxs-lookup"><span data-stu-id="aa002-146">In the **Container** slot that contains the buy box module, select the ellipsis (**...**), and then select **Add Module**.</span></span>

1. <span data-ttu-id="aa002-148">Selecteer **Opslaan**, selecteer **Bewerken voltooien** om de sjabloon in te checken en selecteer **Publiceren** om te publiceren.</span><span class="sxs-lookup"><span data-stu-id="aa002-148">Select **Save**, select **Finish editing** to check in the template, and then select **Publish** to publish it.</span></span>
1. <span data-ttu-id="aa002-149">Ga naar **Pagina's** en open een PDP die de PDP-sjabloon gebruikt.</span><span class="sxs-lookup"><span data-stu-id="aa002-149">Go to **Pages**, and open a PDP that uses the PDP template.</span></span> <span data-ttu-id="aa002-150">Maak een PDP als deze nog niet bestaat.</span><span class="sxs-lookup"><span data-stu-id="aa002-150">If a PDP doesn't yet exist, create one.</span></span>

1. <span data-ttu-id="aa002-152">Selecteer in het dialoogvenster **Module toevoegen** de module **Breadcrumb** en selecteer vervolgens **OK**.</span><span class="sxs-lookup"><span data-stu-id="aa002-152">In the **Add Module** dialog box, select the **Breadcrumb** module, and then select **OK**.</span></span>
1. <span data-ttu-id="aa002-153">Selecteer **Koppelingstekst** in het deelvenster Eigenschappen van het **Breadcrumb**-vak onder **Hoofdmap**.</span><span class="sxs-lookup"><span data-stu-id="aa002-153">In the properties pane of the **Breadcrumb** slot, under **Root**, select **Link text**.</span></span>
1. <span data-ttu-id="aa002-154">Voer in het dialoogvenster **Koppelingstekst** de optie **Start** in en selecteer vervolgens onder **Koppelingsdoel** **Een koppeling toevoegen**.</span><span class="sxs-lookup"><span data-stu-id="aa002-154">In the **Link text** dialog box, enter **Home**, and then, under **Link target**, select **Add a link**.</span></span>
1. <span data-ttu-id="aa002-155">Selecteer in het dialoogvenster **Een koppeling toevoegen** een koppeling voor de breadcrumb-hoofdmap en selecteer vervolgens **OK**.</span><span class="sxs-lookup"><span data-stu-id="aa002-155">In the **Add a link** dialog box, select a link for the breadcrumb root, and then select **OK**.</span></span>
1. <span data-ttu-id="aa002-156">Selecteer **Opslaan** en vervolgens **Preview** om de pagina te bekijken.</span><span class="sxs-lookup"><span data-stu-id="aa002-156">Select **Save**, and then select **Preview** to preview the page.</span></span>
1. <span data-ttu-id="aa002-157">Selecteer **Bewerken voltooien** om de sjabloon in te checken en selecteer **Publiceren** om te publiceren.</span><span class="sxs-lookup"><span data-stu-id="aa002-157">Select **Finish editing** to check in the template, and then select **Publish** to publish it.</span></span>
