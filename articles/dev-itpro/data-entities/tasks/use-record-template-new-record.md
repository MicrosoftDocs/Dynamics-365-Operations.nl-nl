# <a name="create-records-by-using-record-templates"></a>Records maken via recordsjablonen

[!include [task guide banner](../../includes/task-guide-banner.md)]

In deze procedure wordt voorgedaan hoe u een nieuwe record kunt maken met behulp van een eerder gedefinieerde recordsjabloon. Om deze procedure te kunnen uitvoeren, moet u eerst de procedure "Een recordsjabloon maken om de invoer van gegevens te vergemakkelijken" uitvoeren.



In deze procedure wordt het demobedrijf USMF gebruikt.

1. Ga naar Vaste activa > Vaste activa > Vaste activa.
2. Klik op Nieuw.
    * U wordt gevraagd een sjabloon te selecteren. Selecteer de sjabloon die met uw zakelijke behoeften overeenkomt.  
3. Zoek en selecteer de gewenste record in de lijst.
4. Klik op OK.
