# <a name="map-a-cost-element-dimension"></a><span data-ttu-id="736ac-103">Een kostenelementdimensie toewijzen</span><span class="sxs-lookup"><span data-stu-id="736ac-103">Map a cost element dimension</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="736ac-104">Een kostencontroller kan deze procedure gebruiken om een kostenelementdimensie toe te wijzen aan een kostenelementdimensie in de MXMF-rechtspersoon.</span><span class="sxs-lookup"><span data-stu-id="736ac-104">A cost controller can use this procedure to map a cost element dimension to a cost element dimension in the MXMF legal entity.</span></span> <span data-ttu-id="736ac-105">Deze registratie gebruikt het USP2-demogegevensbedrijf.</span><span class="sxs-lookup"><span data-stu-id="736ac-105">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="736ac-106">Ga naar Kostprijsboekhouding > Dimensies > Dimensies van kostenelement.</span><span class="sxs-lookup"><span data-stu-id="736ac-106">Go to Cost accounting > Dimensions > Cost element dimensions.</span></span>
2. <span data-ttu-id="736ac-107">Zoek en selecteer de gewenste record in de lijst.</span><span class="sxs-lookup"><span data-stu-id="736ac-107">In the list, find and select the desired record.</span></span>
    * <span data-ttu-id="736ac-108">Selecteer voor dit voorbeeld Kostenelementen.</span><span class="sxs-lookup"><span data-stu-id="736ac-108">For this example, select Cost elements.</span></span>  
3. <span data-ttu-id="736ac-109">Klik op Dimensietoewijzingen.</span><span class="sxs-lookup"><span data-stu-id="736ac-109">Click Dimension mappings.</span></span>
4. <span data-ttu-id="736ac-110">Klik op Toewijzingen van deze dimensie configureren.</span><span class="sxs-lookup"><span data-stu-id="736ac-110">Click Configure mappings from this dimension.</span></span>
5. <span data-ttu-id="736ac-111">Klik op Nieuw.</span><span class="sxs-lookup"><span data-stu-id="736ac-111">Click New.</span></span>
6. <span data-ttu-id="736ac-112">Typ of selecteer een waarde in het veld Naar dimensie.</span><span class="sxs-lookup"><span data-stu-id="736ac-112">In the To dimension field, enter or select a value.</span></span>
    * <span data-ttu-id="736ac-113">Selecteer voor dit voorbeeld MXMF-kostenelementen.</span><span class="sxs-lookup"><span data-stu-id="736ac-113">For this example, select MXMF Cost elements.</span></span>  
7. <span data-ttu-id="736ac-114">Klik op Nieuw.</span><span class="sxs-lookup"><span data-stu-id="736ac-114">Click New.</span></span>
8. <span data-ttu-id="736ac-115">Markeer in de lijst de geselecteerde rij.</span><span class="sxs-lookup"><span data-stu-id="736ac-115">In the list, mark the selected row.</span></span>
9. <span data-ttu-id="736ac-116">Typ of selecteer een waarde in het veld Van dimensielid.</span><span class="sxs-lookup"><span data-stu-id="736ac-116">In the From dimension member field, enter or select a value.</span></span>
    * <span data-ttu-id="736ac-117">Selecteer voor dit voorbeeld dimensielid 606400 Onkosten voor telefoon en fax.</span><span class="sxs-lookup"><span data-stu-id="736ac-117">For this example, select dimension member 606400 Telephone & Fax Expense.</span></span>  
10. <span data-ttu-id="736ac-118">Typ of selecteer een waarde in het veld Tot dimensielid.</span><span class="sxs-lookup"><span data-stu-id="736ac-118">In the To dimension member field, enter or select a value.</span></span>
    * <span data-ttu-id="736ac-119">Selecteer voor dit voorbeeld dimensielid 6001004 Telefono.</span><span class="sxs-lookup"><span data-stu-id="736ac-119">For this example, select dimension member 6001004 Telefono.</span></span>  
11. <span data-ttu-id="736ac-120">Klik op Opslaan.</span><span class="sxs-lookup"><span data-stu-id="736ac-120">Click Save.</span></span>
