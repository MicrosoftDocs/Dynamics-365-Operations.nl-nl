2. <span data-ttu-id="a7997-107">Selecteer **Nieuw**.</span><span class="sxs-lookup"><span data-stu-id="a7997-107">Select **New**.</span></span>
3. <span data-ttu-id="a7997-108">Typ een waarde in het veld **Groep vaste activa**.</span><span class="sxs-lookup"><span data-stu-id="a7997-108">In the **Fixed asset group** field, type a value.</span></span>

6. <span data-ttu-id="a7997-113">Typ of selecteer een waarde in het veld **Boek**.</span><span class="sxs-lookup"><span data-stu-id="a7997-113">In the **Book** field, enter or select a value.</span></span> <span data-ttu-id="a7997-114">Het veld **Afschrijving berekenen** is ingesteld op **Ja**, dus wordt het activumboek in afschrijvingsvoorstellen opgenomen.</span><span class="sxs-lookup"><span data-stu-id="a7997-114">The **Calculate depreciation** field is set to **Yes**, so the asset book will be included in depreciation proposals.</span></span> <span data-ttu-id="a7997-115">Als **Afschrijving berekenen** is ingesteld op **Nee**, worden de activa niet automatisch afgeschreven.</span><span class="sxs-lookup"><span data-stu-id="a7997-115">If **Calculate depreciation** is set to **No**, the asset will not be automatically depreciated.</span></span>
