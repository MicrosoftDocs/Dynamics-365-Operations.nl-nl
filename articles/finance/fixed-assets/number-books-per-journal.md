# <a name="number-of-books-per-journal"></a><span data-ttu-id="190f3-104">Aantal boeken per journaal</span><span class="sxs-lookup"><span data-stu-id="190f3-104">Number of books per journal</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="190f3-105">Dit onderwerp beschrijft de relatie tussen journalen en activaboeken wanneer u een voorstel voor verwerving of afschrijving van vaste activa maakt met behulp van een batchtaak.</span><span class="sxs-lookup"><span data-stu-id="190f3-105">This topic describes the relationship between journals and asset books when you create a fixed asset acquisition or depreciation proposal through a batch job.</span></span> <span data-ttu-id="190f3-106">U kunt het maximumaantal boeken definiëren dat voor elke verwerving en afschrijving wordt opgenomen met behulp van de velden in de sectie **Aantal boeken per journaal** op het tabblad **Algemeen** van de pagina **Parameters voor vaste activa** (**Vaste activa \> Instellingen \> Parameters voor vaste activa**).</span><span class="sxs-lookup"><span data-stu-id="190f3-106">You can define the maximum number of books that are included for each acquisition and for depreciation by using the fields in the **Number of books per journal** section on the **General** tab of the **Fixed assets parameters** page (**Fixed assets \> Setup \> Fixed assets parameters**).</span></span> <span data-ttu-id="190f3-107">Met deze velden kunt u het aantal activaboeken per verwervingsjournaal en afschrijvingsjournaal verdelen.</span><span class="sxs-lookup"><span data-stu-id="190f3-107">These fields let you distribute the number of asset books per acquisition journal and depreciation journal.</span></span>

<span data-ttu-id="190f3-108">Voor een verwervingsvoorstel is de standaardwaarde minimaal 10.000 boeken.</span><span class="sxs-lookup"><span data-stu-id="190f3-108">For an acquisition proposal, the default value is at least 10,000 books.</span></span> <span data-ttu-id="190f3-109">Voor een afschrijvingsvoorstel is de standaardwaarde minimaal 2.000 boeken.</span><span class="sxs-lookup"><span data-stu-id="190f3-109">For a depreciation proposal, the default value is at least 2,000 books.</span></span>

<span data-ttu-id="190f3-110">Als er bijvoorbeeld 4.000 vaste activa zijn en er twee boeken aan elk activum zijn gekoppeld, wordt één boek op de huidige laag geboekt en wordt het andere boek op de belastingslaag geboekt.</span><span class="sxs-lookup"><span data-stu-id="190f3-110">For example, if there are 4,000 fixed assets, and two books are associated with each asset, one book will be posted to the current layer, and the other book will be posted to the tax layer.</span></span> <span data-ttu-id="190f3-111">Als u 4.000 vaste activa verwerft via batchverwerking, bevat de batchtaak waarmee één verwervingsjournaal voor vaste activa wordt gemaakt met 4.000 activaboeken.</span><span class="sxs-lookup"><span data-stu-id="190f3-111">If you acquire 4,000 fixed assets through batch processing, the batch job that creates one fixed asset acquisition journal will contain 4,000 asset books.</span></span>

> [!NOTE]
> <span data-ttu-id="190f3-112">Het journaal dat wordt gemaakt, wordt gebruikt totdat de batchtaak is voltooid.</span><span class="sxs-lookup"><span data-stu-id="190f3-112">The journal that is created will continue to be used until the batch job is completed.</span></span>
>
> <span data-ttu-id="190f3-113">De afgeleide boeken worden niet opgenomen in het maximumaantal boeken per journaal.</span><span class="sxs-lookup"><span data-stu-id="190f3-113">The derived books aren't included in the maximum number of books per journal.</span></span>

<span data-ttu-id="190f3-114">U kunt batchverwerking gebruiken om afschrijving uit te voeren voor dezelfde reeks verworven activa.</span><span class="sxs-lookup"><span data-stu-id="190f3-114">You can use  batch processing to run depreciation for the same set of acquired assets.</span></span> <span data-ttu-id="190f3-115">Met een batchtaak worden bijvoorbeeld twee afschrijvingsjournalen gemaakt die elk uit 2.000 activaboeken bestaan.</span><span class="sxs-lookup"><span data-stu-id="190f3-115">For example, a batch job creates two depreciation journals, each of which consists of 2,000 asset books.</span></span> <span data-ttu-id="190f3-116">Het eerste journaal bevat boeken die zijn gekoppeld aan de vaste activa die zijn genummerd van 1 tot en met 2.000.</span><span class="sxs-lookup"><span data-stu-id="190f3-116">The first journal will contain books that are associated with the fixed assets that are numbered 1 through 2,000.</span></span> <span data-ttu-id="190f3-117">Het tweede journaal bevat boeken die zijn gekoppeld aan de vaste activa die zijn genummerd van 2.001 tot en met 4.000.</span><span class="sxs-lookup"><span data-stu-id="190f3-117">The second journal will then contain books that are associated with the fixed assets that are numbered 2,001 through 4,000.</span></span>

<span data-ttu-id="190f3-118">De batchverwerkingstaak sluit afgesloten boeken uit.</span><span class="sxs-lookup"><span data-stu-id="190f3-118">The batch processing job excludes closed books.</span></span> <span data-ttu-id="190f3-119">In een batchtaak voor afschrijving worden bijvoorbeeld 10 van de eerste 2.000 boeken afgesloten.</span><span class="sxs-lookup"><span data-stu-id="190f3-119">For example, in a batch job for depreciation, 10 of the first 2,000 books are closed.</span></span> <span data-ttu-id="190f3-120">In dit geval bevat het eerste journaal boeken die zijn gekoppeld aan de vaste activa die zijn genummerd van 1 tot en met 2.011.</span><span class="sxs-lookup"><span data-stu-id="190f3-120">In this case, the first journal will contain books that are associated with the fixed assets that are numbered 1 through 2,011.</span></span> <span data-ttu-id="190f3-121">Het tweede journaal bevat boeken die zijn gekoppeld aan de vaste activa die zijn genummerd van 2.012 tot en met 4.000.</span><span class="sxs-lookup"><span data-stu-id="190f3-121">The second journal will then contain books that are associated with the fixed assets that are numbered 2,012 through 4,000.</span></span>

<span data-ttu-id="190f3-122">De limiet van het aantal boeken wordt toegepast als er geen dubbele activa-id's in hetzelfde journaal aanwezig zijn.</span><span class="sxs-lookup"><span data-stu-id="190f3-122">The limit on the number of books is applied if duplicate asset IDs don't exist in the same journal.</span></span> <span data-ttu-id="190f3-123">Als de activa-id echter hetzelfde is als de boek-id, kan het aantal boeken per journaal worden overschreden om de activa-id in hetzelfde journaal te houden.</span><span class="sxs-lookup"><span data-stu-id="190f3-123">However, if the asset ID is the same as the book ID, the number of books per journal can be exceeded to keep the asset ID in the same journal.</span></span>

<span data-ttu-id="190f3-124">Er zijn bijvoorbeeld 5.001 vaste-activa-id's, drie boeken worden gekoppeld aan elke vaste-activa-id en elk activaboek wordt naar dezelfde boekingslaag geboekt.</span><span class="sxs-lookup"><span data-stu-id="190f3-124">For example, there are 5,001 fixed asset IDs, three books are associated with each fixed asset ID, and each asset book is posted to the same posting layer.</span></span> <span data-ttu-id="190f3-125">U voert de afschrijving uit voor drie opeenvolgende maanden zonder samenvatting.</span><span class="sxs-lookup"><span data-stu-id="190f3-125">You run depreciation for three consecutive months, without summarization.</span></span> <span data-ttu-id="190f3-126">Het afschrijvingsjournaal wordt gemaakt via een batch taak created through a batch job, and the system will create seven journals that have 667 fixed asset IDs and three books for each fixed asset ID.</span></span> <span data-ttu-id="190f3-127">Het resultaat is 2.001 boeken.</span><span class="sxs-lookup"><span data-stu-id="190f3-127">The result will be 2,001 books.</span></span> <span data-ttu-id="190f3-128">Over drie maanden zijn er dus 6.003 journaalregels om dezelfde activa-id's in hetzelfde journaal te behouden.</span><span class="sxs-lookup"><span data-stu-id="190f3-128">Therefore, in three months, there will be 6,003 journal lines to ID.</span></span> <span data-ttu-id="190f3-130">Over drie maanden zijn er 2.988 regels.</span><span class="sxs-lookup"><span data-stu-id="190f3-130">In three months, there will be 2,988
