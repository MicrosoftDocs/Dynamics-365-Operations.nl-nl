[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

<span data-ttu-id="40cbe-105">In dit onderwerp wordt uitgelegd hoe u nummerreeksen voor lease-id's maakt.</span><span class="sxs-lookup"><span data-stu-id="40cbe-105">This topic explains how to create number sequences for lease IDs.</span></span> <span data-ttu-id="40cbe-106">Ook wordt uitgelegd hoe unieke id's worden gemaakt die worden gebruikt in het indexherwaarderingsproces.</span><span class="sxs-lookup"><span data-stu-id="40cbe-106">It also explains how to create unique IDs that are used in the index revaluation process.</span></span>

1. <span data-ttu-id="40cbe-107">Ga naar **Activa leasen \> Instellingen \> Parameters voor activa leasen**.</span><span class="sxs-lookup"><span data-stu-id="40cbe-107">Go to **Asset leasing \> Setup \> Asset leasing parameters**.</span></span>
2. <span data-ttu-id="40cbe-108">Selecteer het tabblad **Nummerreeksen** aan de zijkant.</span><span class="sxs-lookup"><span data-stu-id="40cbe-108">Select the **Number sequences** side tab.</span></span>
3. <span data-ttu-id="40cbe-109">Selecteer **Nummerreeksen** in de zijbalk.</span><span class="sxs-lookup"><span data-stu-id="40cbe-109">Select **Number sequences** in the side bar.</span></span>
4. <span data-ttu-id="40cbe-110">Selecteer de nummerreeks voor de verwijzing **Lease-id**.</span><span class="sxs-lookup"><span data-stu-id="40cbe-110">Select the number sequence for the **Lease ID** reference.</span></span> <span data-ttu-id="40cbe-111">Deze nummerreeks wordt gebruikt om de unieke id voor elke lease te genereren.</span><span class="sxs-lookup"><span data-stu-id="40cbe-111">This number sequence will be used to generate the unique identifier for each lease.</span></span>
5. <span data-ttu-id="40cbe-112">Selecteer de nummerreeks voor de verwijzing **Proces-id**.</span><span class="sxs-lookup"><span data-stu-id="40cbe-112">Select the number sequence for the **Process ID** reference.</span></span> <span data-ttu-id="40cbe-113">Deze nummerreeks wordt gebruikt om de unieke id voor elk indexherwaarderingsproces te genereren.</span><span class="sxs-lookup"><span data-stu-id="40cbe-113">This number sequence will be used to generate the unique identifier for each index revaluation process.</span></span>
6. <span data-ttu-id="40cbe-114">Selecteer de nummerreeks voor de verwijzing **Id van beëindigingsvoorstel**.</span><span class="sxs-lookup"><span data-stu-id="40cbe-114">Select the number sequence for the **Termination Proposal ID** reference.</span></span> <span data-ttu-id="40cbe-115">Deze nummerreeks wordt gebruikt om de unieke id voor elk beëindigingsvoorstel te genereren.</span><span class="sxs-lookup"><span data-stu-id="40cbe-115">This number sequence will be used to generate the unique identifier for each termination proposal.</span></span>
