# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="2d0b9-103">Componentgroepen voor bronbelasting voor het TDS-belastingtype instellen</span><span class="sxs-lookup"><span data-stu-id="2d0b9-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="2d0b9-104">In dit onderwerp wordt uitgelegd hoe u componentgroepen voor bronbelasting, zoals **Huur** en **Contractant**, in moet stellen voor het belastingtype TDS (belasting ingehouden op bron).</span><span class="sxs-lookup"><span data-stu-id="2d0b9-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="2d0b9-105">Ga naar **Belasting \> Instellen \> Bronbelasting \> Componentgroepen voor bronbelasting**.</span><span class="sxs-lookup"><span data-stu-id="2d0b9-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="2d0b9-106">[![Pagina Componentgroepen voor bronbelasting](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="2d0b9-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="2d0b9-107">Selecteer in het veld **Belastingtype** de optie **TDS** om componentgroepen voor bronbelasting in te stellen voor het belastingtype TDS.</span><span class="sxs-lookup"><span data-stu-id="2d0b9-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="2d0b9-108">Selecteer in het actievenster **Nieuw** om een regel te maken.</span><span class="sxs-lookup"><span data-stu-id="2d0b9-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="2d0b9-109">Voer in het veld **Componentgroep voor bronbelasting** een naam in voor de TDS-componentgroep.</span><span class="sxs-lookup"><span data-stu-id="2d0b9-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="2d0b9-110">Voer in het veld **Beschrijving** een beschrijving in.</span><span class="sxs-lookup"><span data-stu-id="2d0b9-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="2d0b9-111">Selecteer op het sneltabblad **Algemeen** in het veld **Status** de status van de TDS-componentgroep.</span><span class="sxs-lookup"><span data-stu-id="2d0b9-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="2d0b9-112">De opties zijn **Ingezetene** en **Niet-ingezetene**.</span><span class="sxs-lookup"><span data-stu-id="2d0b9-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="2d0b9-113">Geef in het veld **Sectiecode** de sectiecode op die op de TDS-componentgroep wordt toegepast.</span><span class="sxs-lookup"><span data-stu-id="2d0b9-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="2d0b9-114">Sluit de pagina.</span><span class="sxs-lookup"><span data-stu-id="2d0b9-114">Close the page.</span></span>
