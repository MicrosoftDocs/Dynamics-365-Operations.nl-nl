# <a name="threshold-limit-and-exception-threshold-limit"></a><span data-ttu-id="752c2-103">Drempellimiet en uitzonderingsdrempellimiet</span><span class="sxs-lookup"><span data-stu-id="752c2-103">Threshold limit and exception threshold limit</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="752c2-104">In dit onderwerp worden de drempel- en uitzonderingslimieten voor TDS (belasting ingehouden op bron) beschreven.</span><span class="sxs-lookup"><span data-stu-id="752c2-104">This topic describes the threshold and exception limits for Tax Deducted at Source (TDS).</span></span> <span data-ttu-id="752c2-105">TDS op facturen en betalingen wordt altijd berekend rekening houdend met de drempellimiet en uitzonderingsdrempellimiet die gedefinieerd zijn voor de TDS-belastingcomponenten op de pagina **Bronbelastingcomponenten**.</span><span class="sxs-lookup"><span data-stu-id="752c2-105">The TDS on invoices and payments is always calculated considering the threshold limit and exception threshold limit defined for the TDS tax components on the **Withholding tax components** page.</span></span> <span data-ttu-id="752c2-106">De TDS-belastingcomponenten zijn gekoppeld aan TDS-belastingcodes die zijn opgenomen in de TDS-belastinggroepen.</span><span class="sxs-lookup"><span data-stu-id="752c2-106">The TDS tax components are attached to TDS tax codes, which are included in the TDS tax groups.</span></span> <span data-ttu-id="752c2-107">De TDS-belastinggroepen worden gekoppeld aan leveranciers en klanten om TDS te berekenen op factuurniveau of betalingsniveau.</span><span class="sxs-lookup"><span data-stu-id="752c2-107">The TDS tax groups are attached to vendors and customers to calculate TDS at the invoice-level or payment-level.</span></span>

<span data-ttu-id="752c2-108">TDS wordt berekend als het bedrag voor een transactie of de cumulatieve transacties die met een specifieke TDS-groep voor een leverancier zijn geboekt, de drempellimiet overschrijdt die is opgegeven op de pagina **Bronbelastingcomponenten**.</span><span class="sxs-lookup"><span data-stu-id="752c2-108">TDS is calculated if the amount for a transaction or the cumulative transactions posted with a specific TDS group for a vendor exceeds the threshold limit specified on the **Withholding tax components** page.</span></span> <span data-ttu-id="752c2-109">TDS wordt pas berekend als het cumulatieve transactiebedrag de opgegeven drempellimiet overschrijdt.</span><span class="sxs-lookup"><span data-stu-id="752c2-109">TDS will not be calculated until the cumulative transaction amount exceeds the specified threshold limit.</span></span>

<span data-ttu-id="752c2-110">Als er een uitzonderingsdrempellimiet is opgegeven samen met de drempellimiet voor een TDS-onderdeel, wordt TDS berekend wanneer een specifiek transactiebedrag de uitzonderingsdrempellimiet overschrijdt, zelfs als het cumulatieve transactiebedrag de opgegeven drempellimiet niet overschrijdt.</span><span class="sxs-lookup"><span data-stu-id="752c2-110">If an exception threshold limit is specified along with the threshold limit for a TDS component, TDS is calculated when a specific transaction amount exceeds the exception threshold limit even if the cumulative transaction amount does not exceed the specified threshold limit.</span></span>
