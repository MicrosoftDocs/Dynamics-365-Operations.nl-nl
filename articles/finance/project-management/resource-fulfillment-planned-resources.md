# <a name="resource-fulfillment-for-planned-resources"></a><span data-ttu-id="d82d6-103">Resourcevoorziening voor geplande resources</span><span class="sxs-lookup"><span data-stu-id="d82d6-103">Resource fulfillment for planned resources</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="d82d6-104">Een projectmanager kan vereiste resourcerollen voor een project plannen.</span><span class="sxs-lookup"><span data-stu-id="d82d6-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="d82d6-105">De resourcemanager ziet deze geplande resources als aanvragen op de pagina **Resourcevervulling** en kan werkelijke resources toewijzen.</span><span class="sxs-lookup"><span data-stu-id="d82d6-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="d82d6-106">Op de pagina **Alle projecten** selecteert u het project **XYZ-upgrade Fase 2**.</span><span class="sxs-lookup"><span data-stu-id="d82d6-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="d82d6-107">Selecteer **Project** en **Bewerken**.</span><span class="sxs-lookup"><span data-stu-id="d82d6-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="d82d6-108">Selecteer op het tabblad **Projectteam en planning** de optie **Toevoegen**.</span><span class="sxs-lookup"><span data-stu-id="d82d6-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="d82d6-109">Selecteer in het dialoogvenster **Rollen toevoegen** de rol **Softwareontwikkelaar**.</span><span class="sxs-lookup"><span data-stu-id="d82d6-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="d82d6-110">Selecteer **Maken** en sluit de projectpagina.</span><span class="sxs-lookup"><span data-stu-id="d82d6-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="d82d6-111">Selecteer op de pagina **Bronafhandeling** de optie **Softwareontwikkelaar 1** voor het project **XYZ-upgrade Fase 2**.</span><span class="sxs-lookup"><span data-stu-id="d82d6-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="d82d6-112">Selecteer een werknemer en selecteer **Toewijzen**.</span><span class="sxs-lookup"><span data-stu-id="d82d6-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="d82d6-113">Controleer of de regel **Softwareontwikkelaar 1** is verwijderd uit het project **XYZ-upgrade Fase 2**.</span><span class="sxs-lookup"><span data-stu-id="d82d6-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="d82d6-114">Controleer op het tabblad **Projectteam en planning** voor het project **XYZ-upgrade Fase 2** of de werknemer die u in de vorige stap hebt geselecteerd, is toegevoegd als **Softwareontwikkelaar**.</span><span class="sxs-lookup"><span data-stu-id="d82d6-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
