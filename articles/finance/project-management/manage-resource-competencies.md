## <a name="set-up-competencies-certificates"></a><span data-ttu-id="8dccc-112">Competenties configureren: getuigschriften</span><span class="sxs-lookup"><span data-stu-id="8dccc-112">Set up competencies: Certificates</span></span>

1. <span data-ttu-id="8dccc-113">Selecteer op de lijstpagina **Werknemers** de regel van de werknemer waarvoor u getuigschriftgegevens wilt toevoegen.</span><span class="sxs-lookup"><span data-stu-id="8dccc-113">On the **Workers** list page, select the line for the worker to add certificate information for.</span></span>
2. <span data-ttu-id="8dccc-114">Selecteer in het actievenster op het tabblad **Werknemer** in de groep **Competenties** de optie **Getuigschriften**.</span><span class="sxs-lookup"><span data-stu-id="8dccc-114">On the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Certificates**.</span></span>

4. <span data-ttu-id="8dccc-116">In het veld **Begindatum** selecteert u **1-10-2015** en vervolgens selecteert u **Opslaan**.</span><span class="sxs-lookup"><span data-stu-id="8dccc-116">In the **Start date** field, select **10/1/2015**, and select **Save**.</span></span>

## <a name="set-up-competencies-skills"></a><span data-ttu-id="8dccc-117">Vaardigheden instellen: vaardigheden</span><span class="sxs-lookup"><span data-stu-id="8dccc-117">Set up competencies: Skills</span></span>

2. <span data-ttu-id="8dccc-120">Selecteer **Nieuw**.</span><span class="sxs-lookup"><span data-stu-id="8dccc-120">Select **New**.</span></span>
3. <span data-ttu-id="8dccc-121">Selecteer in het veld **Vaardigheid** de waarde **Projectbeheer**.</span><span class="sxs-lookup"><span data-stu-id="8dccc-121">In the **Skill** field, select **Project management**.</span></span>
4. <span data-ttu-id="8dccc-122">Selecteer in het veld **Niveau** de waarde **5 Expert**.</span><span class="sxs-lookup"><span data-stu-id="8dccc-122">In the **Level** field, select **5 Expert**.</span></span>

6. <span data-ttu-id="8dccc-124">Voer in het veld **Jaren ervaring** de waarde **10** in.</span><span class="sxs-lookup"><span data-stu-id="8dccc-124">In the **Years of experience** field, enter **10**.</span></span>
7. <span data-ttu-id="8dccc-125">Selecteer **Opslaan** en sluit de pagina.</span><span class="sxs-lookup"><span data-stu-id="8dccc-125">Select **Save**, and then close the page.</span></span>
