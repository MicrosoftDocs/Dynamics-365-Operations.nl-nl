<span data-ttu-id="b2574-104">Gebruik deze taakbegeleider om een aanmaningsreeks te maken.</span><span class="sxs-lookup"><span data-stu-id="b2574-104">Use this task guide to create a collection letter sequence.</span></span> <span data-ttu-id="b2574-105">Bij deze taak wordt het demobedrijf USMF gebruikt.</span><span class="sxs-lookup"><span data-stu-id="b2574-105">This task uses the USMF demo company.</span></span>

1. <span data-ttu-id="b2574-106">Ga in het navigatievenster naar **Modules > Crediteringen en aanmaningen > Instellen > Aanmaningsreeks instellen**.</span><span class="sxs-lookup"><span data-stu-id="b2574-106">In the Navigation pane, go to **Modules > Credit and collections > Setup > Set up collection letter sequence**.</span></span>
2. <span data-ttu-id="b2574-107">Klik op **Nieuw**.</span><span class="sxs-lookup"><span data-stu-id="b2574-107">Click **New**.</span></span>
3. <span data-ttu-id="b2574-108">Voer in het veld **Aanmaningsreeks** een volgorde-id in die de volgorde aangeeft.</span><span class="sxs-lookup"><span data-stu-id="b2574-108">In the **Collection letter sequence** field, enter a sequence ID that will represent the sequence.</span></span> <span data-ttu-id="b2574-109">Deze wordt gebruikt bij het instellen van een boekingsprofiel.</span><span class="sxs-lookup"><span data-stu-id="b2574-109">It will be used when you set up a posting profile.</span></span>
4. <span data-ttu-id="b2574-110">Typ een waarde in het veld **Beschrijving**.</span><span class="sxs-lookup"><span data-stu-id="b2574-110">In the **Description** field, type a value.</span></span>  <span data-ttu-id="b2574-111">De betalingsvoorwaarden zijn optioneel.</span><span class="sxs-lookup"><span data-stu-id="b2574-111">The terms of payment is optional.</span></span> <span data-ttu-id="b2574-112">Als u hier een waarde invoert, gebruikt de factuur voor de aanmaningskosten deze betalingsvoorwaarden in plaats van de betalingsvoorwaarden die zijn opgeslagen met de klant.</span><span class="sxs-lookup"><span data-stu-id="b2574-112">If you enter a value here, the collection letter fee invoice will use these terms of payment instead of the terms of payment stored with the customer.</span></span>  
5. <span data-ttu-id="b2574-113">Selecteer in het veld **Aanmaningscode** de code voor de eerste aanmaning die u wilt verzenden.</span><span class="sxs-lookup"><span data-stu-id="b2574-113">In the **Collection letter code** field, select the code for the first collection letter that you want to send.</span></span> <span data-ttu-id="b2574-114">De eerste aanmaning wordt gemaakt aan de hand van de vervaldatum op de factuur, die waarde die u invoert voor de respijtperiode in het veld Dagen op deze regel en andere gegevens die u in deze regel invoert.</span><span class="sxs-lookup"><span data-stu-id="b2574-114">The first collection letter is created according to the due date on the invoice, the value that you enter for the grace period in the Days field on this line, and other information that you enter on this line.</span></span>  
6. <span data-ttu-id="b2574-115">Typ een waarde in het veld **Beschrijving**.</span><span class="sxs-lookup"><span data-stu-id="b2574-115">In the **Description** field, type a value.</span></span> <span data-ttu-id="b2574-116">De valuta voor de bijzondere kosten is standaard in de valuta van de klant.</span><span class="sxs-lookup"><span data-stu-id="b2574-116">The currency for the fee defaults to the customer currency.</span></span> <span data-ttu-id="b2574-117">Deze valutacode kan verschillen van de factuurvaluta.</span><span class="sxs-lookup"><span data-stu-id="b2574-117">This currency code can be different than the invoice currency.</span></span>  
7. <span data-ttu-id="b2574-118">Klik op **Toevoegen** om de volgende aanmaning toe te voegen die in de reeks wordt verzonden.</span><span class="sxs-lookup"><span data-stu-id="b2574-118">Click **Add** to add the next collection letter that will be sent in the sequence.</span></span> <span data-ttu-id="b2574-119">In veel gevallen is de eerste aanmaning enkel een waarschuwing.</span><span class="sxs-lookup"><span data-stu-id="b2574-119">In many cases, the first collection letter is just a warning.</span></span> <span data-ttu-id="b2574-120">U kunt indien nodig bijzondere kosten toevoegen.</span><span class="sxs-lookup"><span data-stu-id="b2574-120">You can add fees if needed.</span></span>  
8. <span data-ttu-id="b2574-121">Selecteer in het veld van de aanmaningscode de volgende aanmaning die wordt verzonden in de reeks.</span><span class="sxs-lookup"><span data-stu-id="b2574-121">In the collection letter code field, select the next collection letter that will be sent in the sequence.</span></span>
9. <span data-ttu-id="b2574-122">Typ een waarde in het veld **Beschrijving**.</span><span class="sxs-lookup"><span data-stu-id="b2574-122">In the **Description** field, type a value.</span></span>
10. <span data-ttu-id="b2574-123">Selecteer in het veld **Hoofdrekening** de opbrengstrekening die voor bijzondere kosten wordt gebruikt.</span><span class="sxs-lookup"><span data-stu-id="b2574-123">In the **Main account** field, select the revenue account that will be used for fees.</span></span>
11. <span data-ttu-id="b2574-124">Voer de kosten in die worden doorberekend bij het maken van deze aanmaning.</span><span class="sxs-lookup"><span data-stu-id="b2574-124">Enter the fee that will be charged when this collection letter is posted.</span></span>

13. <span data-ttu-id="b2574-127">Klik in de lijst op de koppeling in de geselecteerde rij.</span><span class="sxs-lookup"><span data-stu-id="b2574-127">In the list, click the link in the selected row.</span></span>
14. <span data-ttu-id="b2574-128">Voer in het veld **Minimum achterstallig saldo** het minimale achterstallige saldo in dat vereist is voordat een aanmaning wordt verzonden.</span><span class="sxs-lookup"><span data-stu-id="b2574-128">In the **Minimum overdue balance** field, enter the minimum overdue balance required before a collection letter is sent.</span></span>
15. <span data-ttu-id="b2574-129">Voer in het veld **Dagen** het aantal respijtdagen in dat u toestaat.</span><span class="sxs-lookup"><span data-stu-id="b2574-129">In the **Days** field, enter the number of grace days that you will allow.</span></span> <span data-ttu-id="b2574-130">Dit is het aantal dagen na de vervaldatum waarop een aanmaning kan worden gegenereerd.</span><span class="sxs-lookup"><span data-stu-id="b2574-130">This is the number of days after the due date that a collection letter can be generated.</span></span> <span data-ttu-id="b2574-131">Welke vervaldatum voor de berekening wordt gebruikt, is afhankelijk van de positie van de aanmaning in de aanmaningsreeks:</span><span class="sxs-lookup"><span data-stu-id="b2574-131">The due date that is used for the calculation depends on the position of the collection letter in the collection letter sequence:</span></span>
    - <span data-ttu-id="b2574-132">De respijtperiode voor aanmaning 1 is gerelateerd aan de vervaldatum op de factuur.</span><span class="sxs-lookup"><span data-stu-id="b2574-132">The grace period for collection letter 1 is relative to the due date on the invoice.</span></span>

18. <span data-ttu-id="b2574-137">Typ een waarde in het veld **Beschrijving**.</span><span class="sxs-lookup"><span data-stu-id="b2574-137">In the **Description** field, type a value.</span></span>
19. <span data-ttu-id="b2574-138">Geef in het veld **Hoofdrekening** de gewenste waarden op.</span><span class="sxs-lookup"><span data-stu-id="b2574-138">In the **Main account** field, specify the desired values.</span></span>

22. <span data-ttu-id="b2574-141">Klik in de lijst op de koppeling in de geselecteerde rij.</span><span class="sxs-lookup"><span data-stu-id="b2574-141">In the list, click the link in the selected row.</span></span>
23. <span data-ttu-id="b2574-142">Voer in het veld **Minimum achterstallig saldo** een getal in.</span><span class="sxs-lookup"><span data-stu-id="b2574-142">In the **Minimum overdue balance** field, enter a number.</span></span>
24. <span data-ttu-id="b2574-143">Voer in het veld **Dagen** een getal in.</span><span class="sxs-lookup"><span data-stu-id="b2574-143">In the **Days** field, enter a number.</span></span>

27. <span data-ttu-id="b2574-147">Voer de tekst in die op de aanmaning moet worden weergegeven voor de geselecteerde aanmaningscode.</span><span class="sxs-lookup"><span data-stu-id="b2574-147">Enter the text to appear on the collection letter for the selected collection letter code.</span></span> <span data-ttu-id="b2574-148">Deze tekst kunt u in meerdere talen vertalen via het menu Vertalingen boven het notitievakje.</span><span class="sxs-lookup"><span data-stu-id="b2574-148">You can translate this text in to multiple languages using the Translations menu above the note box.</span></span>
