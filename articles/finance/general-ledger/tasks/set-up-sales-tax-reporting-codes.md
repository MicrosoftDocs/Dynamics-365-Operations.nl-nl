# <a name="set-up-sales-tax-reporting-codes"></a><span data-ttu-id="4a8ce-103">Btw-aangiftecodes instellen</span><span class="sxs-lookup"><span data-stu-id="4a8ce-103">Set up sales tax reporting codes</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="4a8ce-104">De btw-aangiftecodes verwijzen naar een veldnummer dat wordt vermeld in een btw-rapport.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-104">The Sales tax reporting codes refer to a field number that's listed on a sales tax report.</span></span> <span data-ttu-id="4a8ce-105">Deze worden gebruikt voor landspecifieke rapportindelingen.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-105">They are used on country-specific report layouts.</span></span> <span data-ttu-id="4a8ce-106">Ze worden ook gebruikt op het rapport met btw-betaling per code.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-106">They're also used on the Sales tax payment by code report.</span></span> <span data-ttu-id="4a8ce-107">In dat rapport worden btw-bedragen weergegeven voor een vereffeningsperiode die voor elke rapportagecode wordt samengevat.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-107">That report shows sales tax amounts for a settlement period summarized for each reporting code.</span></span> <span data-ttu-id="4a8ce-108">Nadat u de btw-aangiftecodes hebt gemaakt, kunt u naar deze codes verwijzen op de sneltabbladen Rapport instellen, die u kunt vinden op de pagina **Btw-code**.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-108">After you create Sales tax reporting codes, you can refer to those codes on the Report setup FastTabs, which you can access from the **Sales tax code** page.</span></span> 

<span data-ttu-id="4a8ce-109">Bij deze registratie wordt het demobedrijf DEMF gebruikt.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-109">This recording uses the DEMF demo company.</span></span>

1. <span data-ttu-id="4a8ce-110">Ga in het **navigatievenster** naar **Belasting > Instellingen > Btw > Btw-aangiftecodes**.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-110">In the **Navigation pane**, go to **Tax > Setup > Sales tax > Sales tax reporting codes**.</span></span>
2. <span data-ttu-id="4a8ce-111">Klik op **Nieuw**.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-111">Click **New**.</span></span>
3. <span data-ttu-id="4a8ce-112">Selecteer de rapportindeling waarbij de aangiftecode hoort.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-112">Select the report layout that the reporting code belongs to.</span></span> <span data-ttu-id="4a8ce-113">Deze indeling wordt gebruikt om de beschikbare aangiftecodes voor een btw-code te filteren.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-113">This layout is used to filter the available reporting codes for a sales tax code.</span></span> <span data-ttu-id="4a8ce-114">Elke btw-code behoort tot een vereffeningsperiode die bij een belastingdienst hoort die een rapportindeling gebruikt.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-114">Each sales tax code belongs to a settlement period, which belongs to a Sales tax authority, which uses a report layout.</span></span>  
4. <span data-ttu-id="4a8ce-115">Voer een getal in het veld **Aangiftecode** in.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-115">In the **Reporting code** field, enter a number.</span></span>
5. <span data-ttu-id="4a8ce-116">Voer in het veld **Rapporttekst** een omschrijving in voor weergave in rapporten.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-116">In the **Report text** field, enter a description to display on reports.</span></span>
6. <span data-ttu-id="4a8ce-117">Voer in het veld **Korte omschrijving** een omschrijving voor interne doelen in.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-117">In the **Brief description** field, enter a description for internal purposes.</span></span>
7. <span data-ttu-id="4a8ce-118">Klik op **Opslaan**.</span><span class="sxs-lookup"><span data-stu-id="4a8ce-118">Click **Save**.</span></span>
