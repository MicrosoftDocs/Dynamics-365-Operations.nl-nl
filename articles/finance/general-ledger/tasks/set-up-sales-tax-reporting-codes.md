[!include [banner](../../includes/banner.md)]

<span data-ttu-id="fc5d3-104">De btw-aangiftecodes verwijzen naar een veldnummer dat wordt vermeld in een btw-rapport.</span><span class="sxs-lookup"><span data-stu-id="fc5d3-104">The Sales tax reporting codes refer to a field number that's listed on a sales tax report.</span></span> <span data-ttu-id="fc5d3-105">Deze worden gebruikt voor landspecifieke rapportindelingen.</span><span class="sxs-lookup"><span data-stu-id="fc5d3-105">They are used on country-specific report layouts.</span></span> <span data-ttu-id="fc5d3-106">Ze worden ook gebruikt op het rapport met btw-betaling per code.</span><span class="sxs-lookup"><span data-stu-id="fc5d3-106">They're also used on the Sales tax payment by code report.</span></span> <span data-ttu-id="fc5d3-107">In dat rapport worden btw-bedragen weergegeven voor een vereffeningsperiode die voor elke rapportagecode wordt samengevat.</span><span class="sxs-lookup"><span data-stu-id="fc5d3-107">That report shows sales tax amounts for a settlement period summarized for each reporting code.</span></span> <span data-ttu-id="fc5d3-108">Nadat u de btw-aangiftecodes hebt gemaakt, kunt u naar deze codes verwijzen op de sneltabbladen Rapport instellen, die u kunt vinden op de pagina **Btw-code**.</span><span class="sxs-lookup"><span data-stu-id="fc5d3-108">After you create Sales tax reporting codes, you can refer to those codes on the Report setup FastTabs, which you can access from the **Sales tax code** page.</span></span> 

<span data-ttu-id="fc5d3-109">Bij deze registratie wordt het demobedrijf DEMF gebruikt.</span><span class="sxs-lookup"><span data-stu-id="fc5d3-109">This recording uses the DEMF demo company.</span></span>

1. <span data-ttu-id="fc5d3-110">Ga in het **navigatievenster** naar **Belasting > Instellingen > Btw > Btw-aangiftecodes**.</span><span class="sxs-lookup"><span data-stu-id="fc5d3-110">In the **Navigation pane**, go to **Tax > Setup > Sales tax > Sales tax reporting codes**.</span></span>

3. <span data-ttu-id="fc5d3-112">Selecteer de rapportindeling waarbij de aangiftecode hoort.</span><span class="sxs-lookup"><span data-stu-id="fc5d3-112">Select the report layout that the reporting code belongs to.</span></span> <span data-ttu-id="fc5d3-113">Deze indeling wordt gebruikt om de beschikbare aangiftecodes voor een btw-code te filteren.</span><span class="sxs-lookup"><span data-stu-id="fc5d3-113">This layout is used to filter the available reporting codes for a sales tax code.</span></span> <span data-ttu-id="fc5d3-114">Elke btw-code behoort tot een vereffeningsperiode die bij een belastingdienst hoort die een rapportindeling gebruikt.</span><span class="sxs-lookup"><span data-stu-id="fc5d3-114">Each sales tax code belongs to a settlement period, which belongs to a Sales tax authority, which uses a report layout.</span></span>

7. <span data-ttu-id="fc5d3-118">Klik op **Opslaan**.</span><span class="sxs-lookup"><span data-stu-id="fc5d3-118">Click **Save**.</span></span>
