# <a name="mass-financial-period-close"></a><span data-ttu-id="53585-103">Massale sluiting van financiële periode</span><span class="sxs-lookup"><span data-stu-id="53585-103">Mass financial period close</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="53585-104">In dit onderwerp procedure ziet u hoe u een periode in de wachtstand plaatst of definitief afsluit voor meer dan één rechtspersoon tegelijk.</span><span class="sxs-lookup"><span data-stu-id="53585-104">This topic shows how to place a period on hold or permanently close a period or more than one legal entity at a time.</span></span> <span data-ttu-id="53585-105">Daarnaast ziet u hoe u een gebruikersgroep kunt blokkeren voor het boeken naar specifieke modules.</span><span class="sxs-lookup"><span data-stu-id="53585-105">In addition, the task will show how to restrict user group posting to specific modules.</span></span>

1. <span data-ttu-id="53585-106">Ga in het navigatievenster naar **Modules > Grootboek > Afgesloten periode > Grootboekkalenders**.</span><span class="sxs-lookup"><span data-stu-id="53585-106">In the navigation pane, go to **Modules > General ledger > Period close > Ledger calendars**.</span></span> <span data-ttu-id="53585-107">Merk op dat de getoonde lijst van rechtspersonen afhankelijk is van de fiscale kalender die is geselecteerd op de pagina.</span><span class="sxs-lookup"><span data-stu-id="53585-107">Note that the list of legal entities displayed is dependent on the fiscal calendar selected on the page.</span></span> <span data-ttu-id="53585-108">Alleen rechtspersonen die de geselecteerde fiscale kalender gebruiken, worden weergeven.</span><span class="sxs-lookup"><span data-stu-id="53585-108">Only legal entities using the selected fiscal calendar will display.</span></span>
2. <span data-ttu-id="53585-109">Selecteer **Bewerken**.</span><span class="sxs-lookup"><span data-stu-id="53585-109">Select **Edit**.</span></span>
3. <span data-ttu-id="53585-110">Selecteer de periode waarvan u de status wilt wijzigen.</span><span class="sxs-lookup"><span data-stu-id="53585-110">Select the period for which you want to modify the status.</span></span>
4. <span data-ttu-id="53585-111">Selecteer de rechtspersonen waarvan u de status wilt bijwerken.</span><span class="sxs-lookup"><span data-stu-id="53585-111">Select the legal entities for which you want to update the status.</span></span> <span data-ttu-id="53585-112">U kunt alle entiteiten snel selecteren door het selectievakje links boven in het raster in te schakelen.</span><span class="sxs-lookup"><span data-stu-id="53585-112">You can quickly select all legal entities by selecting the check mark on the upper left side of the grid.</span></span>  
5. <span data-ttu-id="53585-113">Selecteer **Moduletoegang bijwerken** om de toegang voor boeken naar een geselecteerde module te definiëren.</span><span class="sxs-lookup"><span data-stu-id="53585-113">Select **Update module access** to define the posting access to a selected module.</span></span> <span data-ttu-id="53585-114">U kunt de modulestatus ook één voor één bijwerken, door de records in het raster te bewerken.</span><span class="sxs-lookup"><span data-stu-id="53585-114">The module status can also be updated one-by-one by editing the records in the grid.</span></span> <span data-ttu-id="53585-115">De knop is handig wanneer u snel meerdere rechtspersoonrecords wilt bijwerken.</span><span class="sxs-lookup"><span data-stu-id="53585-115">The button is useful when you want to quickly update multiple legal entity records.</span></span>  
6. <span data-ttu-id="53585-116">Selecteer in de module **Toepassing** de module waarvan u de status wilt bijwerken.</span><span class="sxs-lookup"><span data-stu-id="53585-116">In the **Application** module, select the module that you want to update the status.</span></span> <span data-ttu-id="53585-117">Klik op **Selecteren**.</span><span class="sxs-lookup"><span data-stu-id="53585-117">Click **Select**.</span></span>
7. <span data-ttu-id="53585-118">Selecteer bij **Toegangsniveau** een waarde: **Alle**, **Geen** of een specifieke gebruikersgroep.</span><span class="sxs-lookup"><span data-stu-id="53585-118">In the **Access** level, select **All**, **None**, or a specific user group.</span></span> <span data-ttu-id="53585-119">Klik op **Selecteren**.</span><span class="sxs-lookup"><span data-stu-id="53585-119">Click **Select**.</span></span> <span data-ttu-id="53585-120">Alle geeft aan dat alle gebruikers met bewerkingsmachtiging naar de module kunnen boeken als de periode is geopend.</span><span class="sxs-lookup"><span data-stu-id="53585-120">All indicates all users with edit access to the module can post if the period is open.</span></span> <span data-ttu-id="53585-121">Geen geeft aan dat gebruikers niet naar de module kunnen boeken als de periode open is.</span><span class="sxs-lookup"><span data-stu-id="53585-121">None indicates that users cannot post to the module if the period is open.</span></span> <span data-ttu-id="53585-122">Een specifieke gebruikersgroep geeft aan dat alleen gebruikers in de groep naar de module mogen boeken als de periode is geopend.</span><span class="sxs-lookup"><span data-stu-id="53585-122">A specific user group indicates only users in the group are able to post to the module if the period is open.</span></span>  
8. <span data-ttu-id="53585-123">Selecteer **Bijwerken**.</span><span class="sxs-lookup"><span data-stu-id="53585-123">Select **Update**.</span></span>
9. <span data-ttu-id="53585-124">Selecteer een andere periode waarvan u de status wilt bijwerken.</span><span class="sxs-lookup"><span data-stu-id="53585-124">Select another period to update the status.</span></span>
10. <span data-ttu-id="53585-125">Selecteer de rechtspersonen waarvan u de periodestatus wilt bijwerken.</span><span class="sxs-lookup"><span data-stu-id="53585-125">Select the legal entites for which you want to update the period status.</span></span>
11. <span data-ttu-id="53585-126">Selecteer **Periodestatus bijwerken** en stel de status in op **In wachtstand**, **Open** of **Definitief afgesloten**.</span><span class="sxs-lookup"><span data-stu-id="53585-126">Select **Update period status** and set the status of **On hold**, **Open**, or **Permanently closed**.</span></span> <span data-ttu-id="53585-127">**Open** geeft aan dat er naar de periode kan worden geboekt, mits de gebruiker toegang heeft.</span><span class="sxs-lookup"><span data-stu-id="53585-127">**Open** indicates the period can be posted to, provided the user has access.</span></span> <span data-ttu-id="53585-128">Bij **In wachtstand** kan er niet worden geboekt naar de periode, maar kan de periode wel worden heropend.</span><span class="sxs-lookup"><span data-stu-id="53585-128">**On hold** means the period cannot be posted to, but the period can be reopened.</span></span> <span data-ttu-id="53585-129">Bij **Definitief afgesloten** is de periode afgesloten en kan deze niet meer worden geopend.</span><span class="sxs-lookup"><span data-stu-id="53585-129">**Permanently closed** means the period is closed and can never be opened.</span></span> <span data-ttu-id="53585-130">Aanpassingen kunnen niet worden geboekt.</span><span class="sxs-lookup"><span data-stu-id="53585-130">Adjustments cannot be posted.</span></span> <span data-ttu-id="53585-131">Het is raadzaam om een periode pas op **Definitief afgesloten** in te stellen wanneer alle correcties en audits zijn voltooid.</span><span class="sxs-lookup"><span data-stu-id="53585-131">We do not recommend setting a period to **Permanently closed** until all adjustments and audits are complete.</span></span>  
12. <span data-ttu-id="53585-132">Selecteer **Bijwerken**.</span><span class="sxs-lookup"><span data-stu-id="53585-132">Select **Update**.</span></span>
