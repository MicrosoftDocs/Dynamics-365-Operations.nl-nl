## <a name="overview"></a><span data-ttu-id="b3498-104">Overzicht</span><span class="sxs-lookup"><span data-stu-id="b3498-104">Overview</span></span>

<span data-ttu-id="b3498-105">Organisaties met medewerkers die per uur worden betaald, kunnen verlof toekennen op basis van gewerkte uren in plaats van de diensttijd bij de organisatie.</span><span class="sxs-lookup"><span data-stu-id="b3498-105">Organizations with hourly employees can award time off based on hours worked instead of tenure with the organization.</span></span> <span data-ttu-id="b3498-106">De gewerkte uren worden doorgaans opgeslagen in een tijd- en aanwezigheidssysteem.</span><span class="sxs-lookup"><span data-stu-id="b3498-106">The hours worked data is typically stored in a time and attendance system.</span></span> 

## <a name="leave-plans"></a><span data-ttu-id="b3498-107">Verlofplannen</span><span class="sxs-lookup"><span data-stu-id="b3498-107">Leave plans</span></span>

<span data-ttu-id="b3498-110">Ga als volgt te werk om een verlofplan met gewerkte uren in te stellen:</span><span class="sxs-lookup"><span data-stu-id="b3498-110">To set up a leave plan to use hours worked, follow these steps:</span></span>

2. <span data-ttu-id="b3498-112">Voer een naam in voor het verlofplan.</span><span class="sxs-lookup"><span data-stu-id="b3498-112">Enter a name for your leave plan.</span></span>
3. <span data-ttu-id="b3498-113">Selecteer de toerekeningsfrequentie voor het plan.</span><span class="sxs-lookup"><span data-stu-id="b3498-113">Select the accrual frequency for the plan.</span></span>
