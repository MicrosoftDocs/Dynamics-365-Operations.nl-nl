# <a name="set-up-fixed-asset-groups"></a><span data-ttu-id="dd0bc-103">Groepen vaste activa instellen</span><span class="sxs-lookup"><span data-stu-id="dd0bc-103">Set up fixed asset groups</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="dd0bc-104">Deze procedure laat zien hoe u een nieuwe vaste-activagroep maakt.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-104">This procedure shows how to create a new fixed asset group.</span></span> <span data-ttu-id="dd0bc-105">Het gebruikt de accountantsrol en demogegevens voor de USMF-rechtspersoon.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-105">It uses the Accountant role and demo data for the USMF legal entity.</span></span>

1. <span data-ttu-id="dd0bc-106">Ga naar Vaste activa > Instellingen > Vaste-activagroepen.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-106">Go to Fixed assets > Setup > Fixed asset groups.</span></span>
2. <span data-ttu-id="dd0bc-107">Klik op Nieuw.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-107">Click New.</span></span>
3. <span data-ttu-id="dd0bc-108">Typ een waarde in het veld Vaste activa.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-108">In the Fixed asset group field, type a value.</span></span>
4. <span data-ttu-id="dd0bc-109">Typ een waarde in het veld Naam.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-109">In the Name field, type a value.</span></span>
    * <span data-ttu-id="dd0bc-110">Vaste activa automatisch nummeren en Nummerreekscode voor de vaste-activagroep overschrijven de instellingen van de vaste-activaparameters.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-110">Autonumber fixed assets and Number sequence code on the Fixed asset group will override the settings on the Fixed assets parameters.</span></span> <span data-ttu-id="dd0bc-111">U kunt het hier aanpassen als de activa in deze vaste-activagroep een andere nummering hebben dan andere groepen.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-111">You can change it here if the assets in this fixed asset group will have different numbering from other groups.</span></span>  
5. <span data-ttu-id="dd0bc-112">Klik op Boeken.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-112">Click Books.</span></span>
6. <span data-ttu-id="dd0bc-113">Typ of selecteer een waarde in het veld Boek.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-113">In the Book field, enter or select a value.</span></span>
    * <span data-ttu-id="dd0bc-114">Het veld Afschrijving berekenen is ingesteld op Ja, dus wordt het activumboek in afschrijvingsvoorstellen opgenomen.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-114">The Calculate depreciation field is set to Yes, so the asset book will be included in depreciation proposals.</span></span> <span data-ttu-id="dd0bc-115">Als Afschrijving berekenen is ingesteld op nee, worden de activa niet automatisch afgeschreven.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-115">If Calculate depreciation is set to No, the asset will not be automatically depreciated.</span></span>  
7. <span data-ttu-id="dd0bc-116">Stel de levensduur van het activum in jaren in.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-116">Set the Service life of the asset, in years.</span></span>
    * <span data-ttu-id="dd0bc-117">Merk op dat de waarde van het veld Afschrijvingsperioden wordt berekend nadat de levensduur is ingesteld.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-117">Note that the Depreciation periods field value is calculated after setting the Service life.</span></span>  
8. <span data-ttu-id="dd0bc-118">Selecteer in het veld Afschrijvingsconventie een optie.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-118">In the Depreciation convention field, select an option.</span></span>
9. <span data-ttu-id="dd0bc-119">Sluit de pagina.</span><span class="sxs-lookup"><span data-stu-id="dd0bc-119">Close the page.</span></span>
