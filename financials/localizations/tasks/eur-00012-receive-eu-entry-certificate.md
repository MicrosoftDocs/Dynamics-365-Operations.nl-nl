<span data-ttu-id="41087-105">Het demobedrijf dat wordt gebruikt om deze procedure te maken is DEMF.</span><span class="sxs-lookup"><span data-stu-id="41087-105">The demo data company used to create this procedure is DEMF.</span></span>

3. <span data-ttu-id="41087-108">Typ een waarde in het veld Invoercertificaat.</span><span class="sxs-lookup"><span data-stu-id="41087-108">In the Entry certificate field, type a value.</span></span>

6. <span data-ttu-id="41087-111">Klik op Opslaan.</span><span class="sxs-lookup"><span data-stu-id="41087-111">Click Save.</span></span>
    * <span data-ttu-id="41087-112">Koppel extra informatie aan het journaal.</span><span class="sxs-lookup"><span data-stu-id="41087-112">Attach additional information to the journal.</span></span>  
    * <span data-ttu-id="41087-113">Koppel bijvoorbeeld een gescande afbeelding van het EU-invoercertificaat aan het journaal.</span><span class="sxs-lookup"><span data-stu-id="41087-113">For example, attach a scanned image of the EU Entry certificate to the journal.</span></span>
