# <a name="create-and-submit-vat-report"></a>Btw-rapport maken en inleveren

[!include[task guide banner](../../includes/task-guide-banner.md)]

Deze procedure laat zien hoe u de btw-aangifte voor Duitsland aanmaakt en indient.

Deze procedure is gemaakt met het demobedrijf DEMF. 

Deze functionaliteit is alleen beschikbaar voor rechtspersonen waarvan het primaire adres in Duitsland is.



1. Ga naar Belastingen > Aangiften > BTW > Btw rapporteren voor vereffeningsperiode.
2. Typ of selecteer een waarde in het veld Vereffeningsperiode.
3. Voer een datum in het veld Begindatum in.
4. Klik op OK.
5. Selecteer 'Ja' in het veld Elektronisch belastingdocument maken.
6. Klik op OK.
