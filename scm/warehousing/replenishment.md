# <a name="replenishment"></a><span data-ttu-id="942ac-103">Aanvulling</span><span class="sxs-lookup"><span data-stu-id="942ac-103">Replenishment</span></span>

[!include[banner](../includes/banner.md)]


<span data-ttu-id="942ac-104">Dit onderwerp beschrijft de aanvullingsstrategieën die beschikbaar zijn voor magazijnen die gebruikmaken van de functionaliteit die beschikbaar is in Magazijnbeheer.</span><span class="sxs-lookup"><span data-stu-id="942ac-104">This topic describes the replenishment strategies that are available for warehouses which use the functionality that is available in Warehouse management.</span></span> <span data-ttu-id="942ac-105">De informatie is niet van toepassing op de magazijnoplossing die in Voorraadbeheer beschikbaar is.</span><span class="sxs-lookup"><span data-stu-id="942ac-105">The information doesn't apply to the warehousing solution that is available in Inventory management.</span></span> <span data-ttu-id="942ac-106">De volgende aanvullingsstrategieën zijn beschikbaar:</span><span class="sxs-lookup"><span data-stu-id="942ac-106">The following replenishment strategies are available:</span></span>

-   <span data-ttu-id="942ac-107">**Aanvulling voor wave-vraag** – Deze strategie maakt aanvullingswerk voor uitgaande orders of ladingen als geen voorraad beschikbaar is wanneer werk wordt gemaakt door de wave.</span><span class="sxs-lookup"><span data-stu-id="942ac-107">**Wave demand replenishment** – This strategy creates replenishment work for outbound orders or loads if inventory isn't available when the wave creates work.</span></span> <span data-ttu-id="942ac-108">Zo kan bijvoorbeeld aanvullingswerk worden gemaakt als de hoeveelheid die is vereist voor een verkooporder niet beschikbaar is wanneer een wave wordt verwerkt.</span><span class="sxs-lookup"><span data-stu-id="942ac-108">For example, replenishment work can be created if the quantity that is required for a sales order isn't available when a wave is processed.</span></span>

-   <span data-ttu-id="942ac-113">**Vraag voor aanvulling laden** – Bij deze strategie wordt de vraag voor verschillende ladingen opgeteld en wordt het aanvullingswerk gemaakt dat is vereist om de relevante orderverzamellocaties te bevoorraden.</span><span class="sxs-lookup"><span data-stu-id="942ac-113">**Load demand replenishment** – This strategy sums the demand for several loads and creates the replenishment work that is required in order to stock the relevant picking locations.</span></span> <span data-ttu-id="942ac-114">Deze strategie helpt te verzekeren dat de ladingen die zijn gemaakt, kunnen worden verzameld in het magazijn nadat ze zijn vrijgegeven.</span><span class="sxs-lookup"><span data-stu-id="942ac-114">This strategy helps to ensure that the loads that are created can be picked in the warehouse after they are released.</span></span>

## <a name="wave-demand-replenishment"></a><span data-ttu-id="942ac-116">Aanvulling voor wave-vraag</span><span class="sxs-lookup"><span data-stu-id="942ac-116">Wave demand replenishment</span></span>

<span data-ttu-id="942ac-117">Bij aanvulling voor wave-vraag wordt aanvullingswerk gemaakt op basis van de vraag als de vereiste hoeveelheid voor productieorders, kanbans, uitgaande orders of ladingen niet beschikbaar is wanneer een wave werk maakt.</span><span class="sxs-lookup"><span data-stu-id="942ac-117">Wave demand replenishment creates replenishment work, based on demand, if the quantity that is required for production orders, kanbans, outbound orders or loads isn't available when a wave creates work.</span></span> <span data-ttu-id="942ac-118">De aanvullingssjabloon bevat informatie over de artikelcriteria, de maateenheid, de toename van de vraag en de locatie.</span><span class="sxs-lookup"><span data-stu-id="942ac-118">The replenishment template contains information about the item criteria, the unit of measure, the demand increment, and the location.</span></span> 

<span data-ttu-id="942ac-119">Locatie-instructies worden gebruikt om te bepalen welke locatie moet worden aangevuld.</span><span class="sxs-lookup"><span data-stu-id="942ac-119">Location directives are used to determine which location should be replenished.</span></span> <span data-ttu-id="942ac-120">U koppelt deze locatie-instructies aan de aanvullingssjabloon met behulp van het veld **Instructiecode**.</span><span class="sxs-lookup"><span data-stu-id="942ac-120">You link these location directives to the replenishment template by using the **Directive code** field.</span></span> <span data-ttu-id="942ac-121">Als het veld **Instructiecode** niet is ingesteld, worden query's gebruikt om te bepalen welke locatie-instructie moet worden gebruikt.</span><span class="sxs-lookup"><span data-stu-id="942ac-121">If the **Directive code** field isn't set, queries are used to determine which location directive should be used.</span></span> <span data-ttu-id="942ac-122">Houd er rekening mee dat als geen instructiecode is opgegeven in de aanvullingssjabloon en de locatie-instructie een instructiecode heeft, wordt de locatie-instructie genegeerd, zelfs als de query op de locatie-instructie correct is.</span><span class="sxs-lookup"><span data-stu-id="942ac-122">Note that if a directive code isn't specified in the replenishment template, and the location directive has a directive code, the location directive will be ignored, even if the query on the location directive is correct.</span></span> <span data-ttu-id="942ac-123">Locatie-instructies voor orderverzamelen worden gebruikt om te bepalen waar u voorraad voor de aanvulling vandaan moet halen.</span><span class="sxs-lookup"><span data-stu-id="942ac-123">Pick location directives are used to determine where to get inventory for the replenishment.</span></span> 

<span data-ttu-id="942ac-124">Naast een sjabloon maken, moet u enkele aanvullingsinstellingen opgeven in de wave-sjabloon.</span><span class="sxs-lookup"><span data-stu-id="942ac-124">In addition to creating a template, you must specify some replenishment settings in the wave template.</span></span> <span data-ttu-id="942ac-125">De wave-sjabloon moet een wave-stap voor aanvulling bevatten die alleen wordt uitgevoerd als toewijzing van een artikel niet succesvol is.</span><span class="sxs-lookup"><span data-stu-id="942ac-125">The wave template should contain a wave step for replenishment that is run only if allocation of an item isn't successful.</span></span> <span data-ttu-id="942ac-126">Deze wave-stap voor aanvulling gebruikt een wave-stapcode om te bepalen welke aanvullingssjabloon moet worden gebruikt.</span><span class="sxs-lookup"><span data-stu-id="942ac-126">This replenishment wave step uses a wave step code to determine which replenishment template should be used.</span></span> <span data-ttu-id="942ac-127">U moet niet alleen een wave-stap voor aanvulling instellen, maar ook ervoor zorgen dat **Aanvullen** is geselecteerd in de sectie **Methoden** van de wave-sjabloon.</span><span class="sxs-lookup"><span data-stu-id="942ac-127">In addition to having a wave step for replenishment, you need to make sure that **Replenish** is selected in the **Methods** section of the wave template.</span></span> 

<span data-ttu-id="942ac-128">De pagina **Aanvullingssjabloon** bevat een selectievakje **Gebruik van niet-gereserveerde hoeveelheid door waveaanvraag toestaan**.</span><span class="sxs-lookup"><span data-stu-id="942ac-128">The **Replenishment template** page includes an **Allow wave demand to use unreserved quantities** check box.</span></span> <span data-ttu-id="942ac-129">Schakel dit selectievakje in als u vraagaanvulling wilt toestaan om niet-gereserveerde hoeveelheden van werk af te trekken van werk dat is gegenereerd op basis van de geselecteerde aanvullingssjabloon.</span><span class="sxs-lookup"><span data-stu-id="942ac-129">You should select this check box if you want to allow demand replenishment to deduct unreserved quantities from work that is generated from the selected replenishment template.</span></span> <span data-ttu-id="942ac-130">Als u sjablonen voor vraagaanvulling in staat wilt stellen om deze logica te gebruiken, moet u dit selectievakje inschakelen voor elke bestaande aanvullingssjabloon.</span><span class="sxs-lookup"><span data-stu-id="942ac-130">To enable demand replenishment templates to use this logic, select this check box for every existing replenishment template.</span></span> <span data-ttu-id="942ac-131">Wanneer de vraagaanvulling in het magazijn wordt geactiveerd, wordt de vraag afgetrokken van het bestaande aanvullingswerk dat niet-gereserveerde hoeveelheden heeft, als het werk afkomstig is uit aanvullingssjablonen waarbij het selectievakje **Gebruik van niet-gereserveerde hoeveelheid door waveaanvraag toestaan** is ingeschakeld.</span><span class="sxs-lookup"><span data-stu-id="942ac-131">When demand replenishment is triggered in the warehouse, it will deduct the demand from existing replenishment work that has unreserved quantities, if the work originates from replenishment templates where the **Allow wave demand to use unreserved quantities** check box is selected.</span></span>


<span data-ttu-id="942ac-132">Vraagaanvulling wordt ondersteund voor verkooporders, transferorders, productieorders en kanbans.</span><span class="sxs-lookup"><span data-stu-id="942ac-132">Demand replenishment is supported for sales orders, transfer orders, production orders, and kanbans.</span></span> 

## <a name="minmax-replenishment"></a><span data-ttu-id="942ac-133">Min/Max aanvulling</span><span class="sxs-lookup"><span data-stu-id="942ac-133">Min/Max replenishment</span></span>

<span data-ttu-id="942ac-140">Houd er rekening mee dat met de strategie van Min/Max-aanvulling geen lege locatie kan worden aangevuld tenzij de locatie is ingesteld als de vaste locatie voor het artikel.</span><span class="sxs-lookup"><span data-stu-id="942ac-140">Note that the Min/Max replenishment strategy can't replenish an empty location unless the location is set as the fixed location for the item.</span></span> <span data-ttu-id="942ac-141">Als de locatie die moet worden aangevuld geen vaste locatie is , kan niet worden bepaald welk artikel moet worden aangevuld.</span><span class="sxs-lookup"><span data-stu-id="942ac-141">If the location that must be replenished isn't a fixed location, it's not possible to determine which item to replenish.</span></span> <span data-ttu-id="942ac-142">Daarom is ten minste enige voorhanden voorraad vereist voordat aanvulling plaatsvindt.</span><span class="sxs-lookup"><span data-stu-id="942ac-142">Therefore, at least some on-hand quantity is required before replenishment occurs.</span></span>
