# <a name="create-a-location-profile"></a><span data-ttu-id="c3bfa-103">Een locatieprofiel maken</span><span class="sxs-lookup"><span data-stu-id="c3bfa-103">Create a location profile</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="c3bfa-104">Elke locatie in het magazijn moet een locatieprofiel hebben dat eraan is gekoppeld en dat de eigenschappen beschrijft van de locatie, bijvoorbeeld of de locatie gemengde artikelen toestaat.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-104">Every location in the warehouse needs to have a location profile associated with it that describes the properties of the location, for example, whether the location allows mixed items.</span></span> <span data-ttu-id="c3bfa-105">In deze procedure wordt een profiel gemaakt voor een locatie die geen controlenummerplaat vereist.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-105">In this procedure we’ll create a profile for a location that doesn’t require license plate control.</span></span> <span data-ttu-id="c3bfa-106">Gemengde artikelen en gemengde voorraadstatussen worden ingeschakeld en cyclustelling is toegestaan.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-106">We’ll enable mixed items, and mixed inventory statuses, and allow cycle counting.</span></span> <span data-ttu-id="c3bfa-107">U kunt deze procedure gebruiken in het demobedrijf USMF.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-107">You can use this procedure in the USMF demo data company.</span></span>

1. <span data-ttu-id="c3bfa-108">Klik op Nieuw.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-108">Click New.</span></span>
2. <span data-ttu-id="c3bfa-109">Typ een waarde in het veld Locatieprofiel-id.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-109">In the Location profile ID field, type a value.</span></span>
3. <span data-ttu-id="c3bfa-110">Typ een waarde in het veld Naam.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-110">In the Name field, type a value.</span></span>
4. <span data-ttu-id="c3bfa-111">Typ of selecteer een waarde in het veld Locatie-indeling.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-111">In the Location format field, enter or select a value.</span></span>
5. <span data-ttu-id="c3bfa-112">Typ of selecteer een waarde in het veld Locatietype.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-112">In the Location type field, enter or select a value.</span></span>
6. <span data-ttu-id="c3bfa-113">Typ of selecteer een waarde in het veld Id dockbeheerprofiel.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-113">In the Dock management profile ID field, enter or select a value.</span></span>
7. <span data-ttu-id="c3bfa-114">Selecteer Ja in het veld Gemengde artikelen toestaan.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-114">Select Yes in the Allow mixed items field.</span></span>
8. <span data-ttu-id="c3bfa-115">Selecteer Ja in het veld Gemengde voorraadstatussen toestaan.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-115">Select Yes in the Allow mixed  inventory statuses field.</span></span>
9. <span data-ttu-id="c3bfa-116">Selecteer Ja in het veld Cyclustelling toestaan.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-116">Select Yes in the Allow cycle counting field.</span></span>
10. <span data-ttu-id="c3bfa-117">Klik op Opslaan.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-117">Click Save.</span></span>
11. <span data-ttu-id="c3bfa-118">Ga naar Magazijnbeheer > Instellingen > Magazijn > Locatieprofielen.</span><span class="sxs-lookup"><span data-stu-id="c3bfa-118">Go to Warehouse management > Setup > Warehouse > Location profiles.</span></span>
