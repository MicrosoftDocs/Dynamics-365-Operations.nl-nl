# <a name="view-inventory-statement-and-kpi-by-cost-object"></a>Voorraadoverzicht en KPI's per kostenobject weergeven

[!include[task guide banner](../../includes/task-guide-banner.md)]

Het bedrijf van de demogegevens dat wordt gebruikt om deze procedure te maken is USMF. Deze procedure is bedoeld voor de kostencontroller.

1. Klik op Kostenadministratie.
2. Klik op Vrijgegeven producten.
3. Gebruik de snelfilter om records te zoeken. Filter bijvoorbeeld op het Veld Artikelnummer met een waarde van 'd0005'.
4. Klik in het actievenster op Kosten beheren.
5. Klik op Kostenobjecten.
6. Klik op Voorraadafschrift en KPI's.
7. Voer een datum in het veld Begindatum in.
8. Voer een datum in het veld Einddatum in.
